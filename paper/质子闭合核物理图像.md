## 二、阶段一：无电荷质子闭合核形成

### 2.1 4-单纯型几何完整化

质子闭合核的几何本体是4-单纯形：

- **5个顶点** → 4-单纯形的顶点
- **10条边** → 4-单纯形的边
  - **3条内部边** → 夸克（内部操作回路）
  - **7条外部边** → 几何闭合的必要条件
- **10个三角形面** → 携带自旋j=1/2
- **5个四面体** → 携带4价intertwiner

**[工作定义P2.1]** 质子闭合核 = 4价j=1/2 SU(2) intertwiner（见[CNT-体系文档 §3.3]）

**intertwiner空间**：4价j=1/2 intertwiner空间维数为2（Catalan数C₂=2），基矢为|I₀⟩和|I₁⟩。

**[反馈点FB-P2.1]** 若高维DCNC实现（如5维→SO(5)）给出更好物理预测，修正4-单纯形为最小实现假设。

### 2.2 初始状态

| 特性 | 描述 |
|------|------|
| 电荷 | 无 |
| 电磁概念 | 无 |
| 再生产 | 极低阶，边holonomy直接做功 |
| 条件债务 | 强相互作用债务（禁闭张力） |
| 边界 | 纯几何边界，无标记分化 |

### 2.3 边界能量量子

质子再生产的原材料/债务通货来自边界能量量子：

$$\text{边界能量量子} = \frac{\text{质子再生产做功}}{\text{在场条件}}$$

这些量子尚未获得任何电磁标记，是纯粹的几何能量单元。

**[物理诠释P2.1]** "边界能量量子"是启发式概念，其严格数学定义待建立（可能与DCNC工作公理2.3的时态化结构相关）。

## 四、阶段三：电荷沉淀 = 边界标记锁定

### 4.1 触发机制（工作猜想）

再生产重复实现 → 路径依赖 → 偏好偏差固化

**[反馈点FB-P4.1]** 若"路径依赖→偏好偏差固化"无法从DCNC工作公理2.8-2.9（选择性历史+锁定机制）推导，修正触发机制描述。

### 4.2 数学表述（依赖CNT-体系文档）

4-单纯形边界在历史路径积分中获得稳定U(1)相位锁定。

**严格数学表述**（参见[CNT-体系文档 §5.1]）：

历史路径积分（HPI）有效映射：
$$R_{\text{eff}} = N_+ (I - \beta e^{i\phi} R_0)^{-1} + N_- (I - \beta e^{-i\phi} R_0)^{-1}$$

其中：
- $\phi = 5\arccos(1/4) \bmod 2\pi$ 是EPRL相位（来自Regge作用量）
- $R_0$ 是再生产映射（3×3矩阵，作用于带电轻子子空间）
- $\beta, N_+, N_-$ 是待从第一性原理推导的参数

**物理诠释**：
- 历史路径积分是对质子闭合核所有可能再生产历史的求和
- 每次再生产引入因子$\beta e^{\pm i\phi} R_0$
- U(1)相位锁定体现在EPRL相位$\phi$的周期性

**[重要声明]** Wilson loop公式$\oint_\Sigma A_\mu dx^\mu = n\cdot e$是标准QED的低能有效描述，在BD-LQG框架中，$A_\mu$需要从intertwiner几何涌现。

### 4.3 质子获电荷（工作猜想）

4-单纯形边界获得稳定U(1)相位 → 历史路径积分中绕数锁定

**[反馈点FB-P4.2]** 若U(1)相位锁定无法从DCNC工作公理严格推导，修正电荷起源诠释。

### 4.4 电荷外溢 = 向下因果（工作猜想）

质子界标记锁定后，对边界外能量量子施加规范性约束。早期电子被迫响应此约束。

$$\text{电子"获得"电荷} = \text{携带质子边界标记的外溢}$$

**因果链**（**工作猜想**）：

```
质子边界标记锁定
    ↓ 对边界外能量量子施加约束
电子被迫响应
    ↓ 电子"获得"电荷
```

**[反馈点FB-P4.3]** 若"向下因果"机制与标准QED的局域规范不变性矛盾，修正诠释或承认此方向需要更精细的数学表述。

### 6.2 基本常数的沉淀起源（工作猜想）

| 常数 | 几何起源 | 状态 |
|------|----------|------|
| $\alpha$ | 4-单纯形二面角 $\Theta = \arccos(1/4)$ 的几何相位 | 工作猜想 |
| $\varepsilon_0$ | 边界标记耦合效率的历史沉淀 | 构建中 |
| $\mu_0$ | 边界张力-响应关系的历史沉淀 | 构建中 |

**[反馈点FB-P6.2]** 若$\varepsilon_0, \mu_0, c$的"历史沉淀"诠释无法从DCNC工作公理推导，降级为纯启发式描述。

### 6.3 精细结构常数的几何推导（工作猜想）

**核心物理原则**（**工作猜想**）：历史沉淀不是凭空发生的，它必然会反映在闭合核的几何结构上。

精细结构常数 $\alpha$ 作为无量纲常数，其演化必须从几何结构上理解：

- $\alpha$ 不是外部参数，而是闭合核几何结构的内禀属性
- 历史沉淀改变了闭合核的几何结构（通过边界标记锁定）
- 因此 $\alpha$ 的演化 = 闭合核几何结构的演化

**严格数学公式**（参见[CNT-体系文档 §5.1]）：

$$\frac{1}{\alpha_0} = \frac{4\pi}{\sin^2\phi}$$

其中 $\phi = 5\Theta \bmod 2\pi = 5\arccos(1/4) \bmod 2\pi$ 是EPRL相位。

**数值结果**：
- $\Theta = \arccos(1/4) \approx 1.3181$ rad
- $\phi = 5\Theta \bmod 2\pi \approx 0.3074$ rad
- $\sin^2\phi = 375/4096 \approx 0.0916$
- $1/\alpha_0 = 16384\pi/375 \approx 137.258$

**与实验值的偏差**：
- 实验值：$1/\alpha \approx 137.036$
- 偏差：$(137.258 - 137.036)/137.036 \approx 0.162\%$

**[重要声明]** 此偏差需要解释，可能来源于：
1. 高阶几何修正
2. 历史路径积分的累积效应
3. DCNC耗散修正（工作公理2.5）

**[反馈点FB-P6.3]** 若0.162%偏差无法从DCNC工作公理解释，分析断裂点（可能是工作公理问题或计算问题）。

**历史沉淀的几何反映**（**工作猜想**）：

1. **边界标记锁定**：4-单纯形边界获得稳定U(1)相位，改变了边界的几何结构
2. **几何结构演化**：从纯几何（无标记）到带标记的几何
3. **几何到物理的映射**：边界标记的几何结构 → U(1)规范对称性 → 电荷量子化 → 精细结构常数

因此，$\alpha$ 的演化是闭合核几何结构演化的直接反映。

## 八、轻子族：改造深度的层级结构

### 8.1 核心命题（工作猜想）

μ子、τ子、电子是质子闭合核再生产改造的**不同深度/能级**产物。

**[反馈点FB-P8.1]** 若轻子质量比无法从"改造深度"框架计算，修正工作猜想5.4或承认此诠释不成立。

### 8.2 层级表

| 轻子  | 改造深度 | 再生产节拍                                          | 寿命                                   | 归宿                                |
| :-- | :--- | :--------------------------------------------- | :----------------------------------- | :-------------------------------- |
| 电子  | 浅层   | $\tau_e = \hbar/m_ec^2 \sim 10^{-21}$ s        | 稳定（$N_{\text{max}} \to \infty$）      | 原子结合                              |
| μ子  | 中层   | $\tau_\mu = \hbar/m_\mu c^2 \sim 10^{-23}$ s   | $2.2$ μs（$N_{\text{max}} \sim 10^6$） | $\to e^- + \bar{\nu}_e + \nu_\mu$ |
| τ子  | 深层   | $\tau_\tau = \hbar/m_\tau c^2 \sim 10^{-25}$ s | $0.3$ ps（$N_{\text{max}} \sim 10^4$） | $\to \mu/e + \nu + \text{强子}$     |

### 8.3 寿命公式（工作猜想）

$$\tau = N_{\text{max}} \times \tau_{\text{reproduction}}$$

其中：
- $N_{\text{max}}$：最大完全再生产周期数（结构稳定性阈值）
- $\tau_{\text{reproduction}}$：单次再生产节拍

**规律**（**工作猜想**）：改造越深 → 质量越大 → 节拍越短 → 完全再生产周期数越少 → 寿命越短

**[反馈点FB-P8.2]** 若寿命公式与实验值不符（考虑中微子质量、CKM矩阵等），修正公式或承认此框架不完整。
