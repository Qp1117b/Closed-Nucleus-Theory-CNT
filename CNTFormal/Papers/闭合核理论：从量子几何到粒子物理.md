## 2 闭合核公理体系与数学实现

### 2.1 公理

CST基于以下五个公理：

**[公理1] (物质公理)** 物质是自我差异化、历史组织的终极存在。

**[公理2] (闭合公理)** 自我差异化必然趋向操作闭合的形成。

**[公理3] (视角公理)** 操作闭合必然伴生结构性内外区分(闭合视角性)。

**[公理4] (债务公理)** 闭合条件不可能完全在场，故必有条件债务——系统必须持续行动以获取不在场的条件，不行动即解体。

**[公理5] (沉淀公理)** 条件债务驱动的选择性在重复行动中产生差异结果，有利于稳定存在的模式被锁定为历史沉淀——事实性倾向规范。

**[注释]** CST的公理不是数学自明真理，而是对物质存在方式的操作性描述。它们的有效性由理论的整体解释力——以及可检验预言的准确性——来验证。

---

### 2.2 闭合核的严格判据

**[定义2.1]** 一个物质结构是闭合核当且仅当同时满足：

| 判据 | 含义 |
|------|------|
| (A) **操作闭合** | 存在一个操作回路，做功筛选/摄取/转化物质/能量/信息，并以维持该回路自身为终点。 |
| (B) **自我指涉** | 回路区分"参与的操作"与"环境扰动"，产生内外之分。 |
| (C) **再生产** | 系统持续生成维持闭合的条件，在流变中保持组织闭合的同一性。 |
| (D) **条件债务** | 闭合条件不完全在场，须持续行动——不行动即解体。 |

**[定理2.1]** 满足四条判据的最小SU(2)量子系统是4价j=1/2 intertwiner。

**证明纲要**:
1. 操作闭合需要至少一个SU(2)不变量：$\sum \vec{J}_i = 0$。单个或两个自旋1/2不能构成非平凡的J=0态(对单一粒子$\vec{J}=0$意味着无自旋；两个自旋1/2的J=0态是唯一的singlet，无内部结构)。三个自旋1/2不能形成J=0态(奇数个自旋1/2的总自旋总是半整数)。四个自旋1/2系统是形成非平凡J=0态空间的最少配置。
2. $4 \times 1/2$的SU(2)张量积分解：$2^{\otimes 4} = 2 \oplus 2 \oplus 2 \oplus 2 \oplus 1 \oplus 1 \oplus 1 \oplus 1 \oplus 1 \oplus 1$(注：实际上$2^{\otimes 4}={\bf 5} \oplus {\bf 3} \oplus {\bf 3} \oplus {\bf 1}$，即自旋3/2(4维)出现1次，自旋1/2(2维)出现3次，自旋0(1维)出现2次。但我们需要的是J=0→$\mathbf{1}$子空间，其中多重度为2。)
3. $\dim\mathcal{H}_{\text{int}}(4,1/2) = C_2 = 2$(Catalan数)。
4. 视角性$\kappa = \dim\mathcal{H}_{\text{int}}/\dim\mathcal{H}_{\text{raw}} = 2/16 = 1/8$。

**[数值事实2.1]** 两个正交的intertwiner基矢(使用标准CG系数构造):

```
|I₀⟩: (1/2,1/2) → 0 ⊗ (1/2,1/2) → 0     (singlet-singlet通道)
|I₁⟩: (1/2,1/2) → 1 ⊗ (1/2,1/2) → 1 → 0 (triplet-triplet通道)
```

验证:
$$\langle I_0|I_0\rangle = 1,\quad \langle I_1|I_1\rangle = 1,\quad \langle I_0|I_1\rangle = 0,\quad J_{\text{total}}|I_{0,1}\rangle = 0$$

### 2.3 质子的闭合核验证

**[推导2.1]** 质子(夸克构成uud, $J=1/2$)满足闭合核所有判据：

* **(A) 操作闭合**: QCD胶子交换维持夸克禁闭。SU(3)$_c$动力学形成了一个操作回路：夸克交换胶子→胶子传递色荷→色闭合(色单态)。回路的终点是维持质子自身的束缚态。
* **(B) 自我指涉**: intertwiners约束$J_{\text{total}}=0$是SU(2)规范不变量。"内"是J=0子空间(闭合核内部动力学)；"外"是J$\neq$0子空间(环境扰动，可导致衰变或散射)。
* **(C) 再生产**: EPRL顶点振幅(见§3)描述了4-单纯形量子跃迁，保持了intertwiner结构在时间演化中的同一性。
* **(D) 条件债务**: $J_{\text{total}}=0$不是自动满足的——需要持续的胶子和海夸克动力学来维护。胶子场停止→夸克分离→解体。

**[重要说明]** 上述"验证"在当前阶段是概念性的:它将CST的操作性描述映射到QCD已知现象上。严格的第一性验证——从闭合核公理出发推导QCD——尚未完成，因为CST目前不具备独立的强耦合计算框架。然而，该映射在后续各节中被证实具有强大的预测力。

### 2.4 6j符号: intertwiner空间的变换矩阵

**[定义2.2]** 6j符号$\{j_1,j_2,j_{12}; j_3,j_4,j_{34}\}$是两个不同耦合方案的变换系数。对于4价j=1/2系统：

$$\begin{Bmatrix} \frac12 & \frac12 & k \\ \frac12 & \frac12 & l \end{Bmatrix}$$

其中$k,l \in \{0,1\}$标记两个配对通道。(0=singlet, 1=triplet)

**[数值事实2.2]** (使用SymPy的wigner_6j函数精确计算):

```python
from sympy.physics.wigner import wigner_6j
from sympy import Rational, N
s6j = lambda a,b,c,d,e,f: float(N(wigner_6j(
    Rational(a,1) if isinstance(a,int) else Rational(a[0],a[1]),
    Rational(b[0],b[1]), Rational(c),
    Rational(d[0],d[1]), Rational(e[0],e[1]), Rational(f,1))))
```

结果:

| 6j符号 | 精确值 | 数值 |
|--------|--------|------|
| $\{1/2,1/2,0;1/2,1/2,0\}$ | $-1/2$ | $-0.5$ |
| $\{1/2,1/2,0;1/2,1/2,1\}$ | $+1/2$ | $+0.5$ |
| $\{1/2,1/2,1;1/2,1/2,0\}$ | $+1/2$ | $+0.5$ |
| $\{1/2,1/2,1;1/2,1/2,1\}$ | $+1/6$ | $+0.166666...$ |

**[定理2.2]** (正交性) 4价intertwiner的6j符号满足：

$$\sum_{l=0}^1 (2l+1)\begin{Bmatrix} \frac12 & \frac12 & k \\ \frac12 & \frac12 & l \end{Bmatrix}^2 = \frac{1}{2k+1}$$

**验证**:

对于$k=0$: $\sum_l (2l+1)\{6j\}^2 = 1.0 = 1/(2\cdot0+1) = 1$ ✓
对于$k=1$: $\sum_l (2l+1)\{6j\}^2 = 0.333... = 1/(2\cdot1+1) = 1/3$ ✓

$$\mathcal{T} = \begin{pmatrix} -1/2 & +1/2 \\ +1/2 & +1/6 \end{pmatrix}$$

## 3 再生产映射与三代轻子的涌现

### 3.1 再生产映射的定义

**[定义3.1]** 再生产映射$\mathcal{R}_0: \mathcal{H}_{\text{int}} \to \mathcal{H}_{\text{int}}$描述闭合核在EPRL顶点作用下的动力学演化。其矩阵元由15j符号的partial trace给出：

$$(\mathcal{R}_0)^{kl} = \sum_{n_A,n_B,n_C} \overline{\langle k | n_D,n_E \rangle_{\text{CG}}} \langle l | n_D,n_E \rangle_{\text{CG}} \sum_{n_D,n_E} 15j(n_A,n_B,n_C,n_D,n_E) \cdot \overline{15j(m_A,m_B,m_C,n_D,n_E)}$$

其中$k,l \in \{0,1,2\}$标记配对态。

### 3.2 三代轻子的几何起源

**[定理3.1]** 3代轻子对应于3维配对态空间。这是4维4-单纯形再生产过程中的结构必然性。

**证明**:

考虑EPRL顶点振幅$A_v$，它耦合了5个四面体(对应5个intertwiners)的边。当质子的4价intertwiner与一个轻子-反轻子对耦合时，系统变为5价intertwiner：

```
质子(4价) + 再生产 → 5价系统
                   ↓
           3条内部边(维持质子结构)
         + 2条外部边(耦合轻子对)
```

这2条外部边$(N_D,N_E)$形成配对态。考虑三个独立的配对基矢：

| 态 | 自旋 | 量子数 | 对应轻子 |
|----|------|--------|----------|
| $|\psi_0\rangle$ | singlet (j=0) | $(N_D,N_E) = (0,1) \otimes (1,0)$ | tau-like (最重) |
| $|\psi_1\rangle$ | triplet m=-1 (j=1) | $(N_D,N_E) = (1,1)$ | mu-like (中间) |
| $|\psi_2\rangle$ | triplet m=+1 (j=1) | $(N_D,N_E) = (0,0)$ | e-like (最轻) |

PAIR态空间的维数为3。**三代轻子是此3维空间的必然数学结果**。不存在第四代轻子的可能，因为$2^{\otimes 2}$的Clebsch-Gordan分解只有singlet和triplet。

**[推导3.1]** CG系数:

$$\langle k|N_D,N_E\rangle = \text{CG}(N_D,N_E \to k)$$

具体值:
$$\langle 0|0,1\rangle = 1/\sqrt{2},\quad \langle 0|1,0\rangle = -1/\sqrt{2}$$
$$\langle 1|1,1\rangle = 1,\quad \langle 2|0,0\rangle = 1$$
其他矩阵元为零。

### 3.3 15j符号的CG收缩计算

**[定义3.2]** 15j(15j符号)描述5个四面体(每个有自旋k_e外腿)组成的4-单纯形的自旋网络配分。对于j=1/2的所有内部边：

$$15j(k_0,k_1,k_2,k_3,k_4) = \sum_{\{m_{ef}=\pm 1/2\}} \prod_{v=0}^4 A_v(k_v; m_{v0},m_{v1},m_{v2},m_{v3})$$

其中$A_v$是第v个顶点的4-spinor到自旋k_v, m=0的CG振幅。

**[数值事实3.1]** 我们实现了直接的CG收缩(2$^{10}$=1024项求和)，使用以下CG系数库：

```python
CG_half = {  # ⟨½,m₁; ½,m₂ | J,M⟩
    (+1,+1,1,+2): 1.0,                # ⟨+,+|1,+1⟩
    (+1,-1,1,0): 1/√2,                # ⟨+,-|1,0⟩
    (-1,+1,1,0): 1/√2,               # ⟨-,+|1,0⟩
    (-1,-1,1,-2): 1.0,               # ⟨-,-|1,-1⟩
    (+1,-1,0,0): 1/√2,               # ⟨+,-|0,0⟩
    (-1,+1,0,0): -1/√2,              # ⟨-,+|0,0⟩(符号)
}
```

结果:

- **非零15j符号数**: 14/32 ($43.75\%$)
- **计算时间**: $\sim 0.13$s (单核Python)
- **求和项**: $2^{10} = 1024$ (每条内边有2种磁量子数m=±1/2)

**15j值按各腿激发数(N_ones = $\sum_e k_e$)分布**:

| N_ones | 个数 | 典型值量级 | 精确代数形式 |
|--------|------|-----------|-------------|

| 2 | 10 | $\sim 0.272$ | |

**[注释]** 15j符号的值是精确有理数(或$\sqrt{n}$的线性组合)，因为CG系数是精确代数数。Wolfram Mathematica的`NineJSymbol`函数的精确代数结果可在`fifteenj_exact.wl`中找到，但其化简形式尚未完全解析。

### 3.4 R$_0$矩阵的构造

**[推导3.2]** R$_0$是3×3的Hermitian矩阵，其矩阵元由15j的partial trace给出：

$$(\mathcal{R}_0)^{kl} = \sum_{n_A,n_B,n_C} \langle k|n_D,n_E\rangle \cdot \overline{\langle l|n_D,n_E\rangle} \cdot w(n_D,n_E)$$

其中权重$w(n_D,n_E) = \sum_{n_A,n_B,n_C} |15j_{\text{EPRL}}(n_A,n_B,n_C,n_D,n_E)|^2$。

**[数值事实3.2]** 使用方案B(相干混合，保留15j的相位信息)得到的R$_0$:

$$
R_0^{\text{(SU(2), EPRL weighted)}} =
\begin{pmatrix}
0.1928 + 0i & 0 & 0 \\
0 & 0.2478 & 0.0904 - 0.0191i \\
0 & 0.0904 + 0.0191i & 0.1920
\end{pmatrix}
$$

**[重要发现]** R$_0$**不是**循环矩阵！

- Singlet态($k=0$)与两个Triplet态($k=1,2$)耦**为零**。
- Triplet态之间有非零耦合($0.0904 \pm 0.0191i$)
- 但Singlet与Triplet之间无耦合。
- 物理含义: EPRL投影破坏了S$_3$循环对称性。

**[开放问题3.2]** Singlet-Triplet解耦的物理原因：是否因为$(N_D,N_E)=(0,1)/(1,0)$通道与$(0,0)/(1,1)$通道在EPRL相位因子下正交？这一解耦是否对应轻子与中微子的某种结构区分？

### 3.5 经验关系: HPI与质量公式

**[经验关系3.1]** 零质量动力学(History Path Integral, HPI)提出质量来自再生产映射的几何级数：

$$R_{\text{eff}}(\theta) = \sum_{n=0}^\infty \beta^n(N_+ e^{in\phi} + N_- e^{-in\phi}) R_0^n$$

闭合形式：
$$R_{\text{eff}} = N_+(I - \beta e^{i\phi} R_0)^{-1} + N_-(I - \beta e^{-i\phi} R_0)^{-1}$$

标量函数形式(假设R$_0$与R$_{\text{eff}}$共用本征态)：
$$f(\mu) = \frac{N_+}{1 - \beta e^{i\phi} \mu} + \frac{N_-}{1 - \beta e^{-i\phi} \mu}$$

**质量公式**:
$$m_k = m_0 |f(\lambda_k)|^2$$

其中$\lambda_k = \text{eig}_k(R_0)$，$m_0$是归一化常数(比例于质子质量)。

**参数**(目前为拟合值):

| 参数 | 值 | 来源 | 状态 |
|------|-----|------|------|
| $\beta$ | $0.79$ | 随机搜索 | **[拟合]** |
| $a$ (R$_0$对角) | $0.737$ | 随机搜索 | **[拟合]** |
| $|b|$ (R$_0$非对角) | $0.448$ | 随机搜索 | **[拟合]** |
| $\arg(b)$ | $-16.6^\circ$ | 随机搜索 | **[拟合]** |

**[注释]** $\beta$在此扮演再生产过程的"耦合常数"角色，其物理意义为单次再生产循环的成功概率幅。$\beta$可能来自IR固定点或自洽条件$m_k/m_p = |f(\lambda_k)|^2$。

**[开放问题3.2]** $\beta$的第一性原理推导。目前所有质量公式都依赖$\beta$作为可调参数。从EPRL顶点振幅出发，$\beta$可能与视图性$\kappa=1/8$有关，或与R$_0$的最大本征值$\max|\lambda_k|$有关。

### 3.6 第一性原理R$_0$的问题

**[重要警示]** 从15j CG收缩直接计算得到的R$_0$本征值**远远不足以产生实验质量等级**:

- **第一性R$_0$本征值**: $|\lambda| = [0.316, 0.193, 0.123]$ (比值: $1 : 0.61 : 0.39$)
- **实验需要**: $|f(\lambda_\tau)| : |f(\lambda_\mu)| : |f(\lambda_e)| = 1 : 0.244 : 0.017$
- **含J=3/2增强(因子4.0)**: $|\lambda| = [0.452, 0.134, 0.046]$ (比值: $1 : 0.29 : 0.10$)
-   **仍远远不够**

**[推论]** 质量等级的大差距($m_\tau/m_e \approx 3500$)意味着当前配对投影方案缺失了关键物理——要么R$_0$在物理的HPI过程中被重整化，要么配对态基矢$|\psi_k\rangle$不是质量本征态的正确零级近似。

**[猜想3.1]** 质量等级可能来自HPI的"奇点放大"效应：当$\beta \cdot |\lambda_k| \to 1$时，$|f(\lambda_k)|$非解析发散。如果物理的R$_0$本征值接近$e^{\pm i\phi}/\beta$的$\beta$参数化奇点，则本征值微小差距可被HPI指数放大。但这需要$|\lambda_k| \sim O(1)$，而目前的15j R$_0$只有$O(0.1)$。

### 3.7 Koide关系的经验接近

**[经验关系3.2]** Koide关系:

$$K = \frac{m_1 + m_2 + m_3}{(\sqrt{m_1} + \sqrt{m_2} + \sqrt{m_3})^2} = \frac{2}{3}$$

在HPI框架中，$m_k \propto |f(\lambda_k)|^2$，Koide变为：

$$K = \frac{\sum_k |f(\lambda_k)|^4}{(\sum_k |f(\lambda_k)|^2)^2}$$

**[数值事实3.3]** 在我们对质量比匹配的$\mu_k$搜索中，找到K∈[0.57,0.69]的解，其中K=2/3附近的解集中在$\beta\sim 0.8$区域。

**[开放问题3.3]** Koide关系($K=2/3=0.666...$)是来自R$_0$的代数结构，还是HPI的动力学近似？目前不明确。

## 4 电荷作为历史沉淀与$\alpha$

### 4.1 Berry相位与拓扑电荷 ⚠️ 待重新推导

**[定义4.1]** 考虑EPRL相位的参数延拓：将固定相位$\phi$替换为可调参数$\theta$：

$$R_{\text{eff}}(\theta) = N_+(I - \beta e^{i\theta} R_0)^{-1} + N_-(I - \beta e^{-i\theta} R_0)^{-1}$$

当$\theta: 0 \to 2\pi$时，$R_{\text{eff}}$的本征态在参数空间中演化。

**[定理4.1]** 如果R$_0$是正规矩阵(R$_0$与R$_0^\dagger$对易)，则$R_{\text{eff}}(\theta)$与R$_0$共用本征态。因此$R_{\text{eff}}$的本征态不依赖$\theta$，Berry相位平凡为零。

**证明**：正规矩阵可对角化为$R_0 = U D U^\dagger$，其中$D$是对角矩阵。则$R_{\text{eff}}(\theta) = U f(D,\theta) U^\dagger$，本征向量$U$不依赖$\theta$。因此Berry联络$A = i\langle\psi|\partial_\theta|\psi\rangle = 0$，Berry相位$\gamma_B = \oint A d\theta = 0$。$\square$

**[重要说明]** 定理4.1表明HPI路径积分形式本身不产生Berry相位。Berry相位必须来自更深层的结构。

**⚠️ [之前版本的错误]** 之前版本提出了矩阵：

$$A(\theta) = \frac{1}{\sqrt{2}} \begin{pmatrix}
-\frac12 & -\frac32 e^{-i\theta} \\[4pt]
\frac12 e^{i\theta} & -\frac12
\end{pmatrix}$$

并声称其本征值为$e^{\pm 2\pi i/3}$，Berry相位为$\pm\pi/2, \pm3\pi/2$。这是**错误**的：

1. $A(\theta)$不是幺正矩阵（$A^\dagger A \neq I$）
2. 本征值$|\lambda| = \sqrt{1/2} \neq 1$，不满足幺正性
3. Racah矩阵的严格本征值是$\pm 1$（标准SU(2) Racah-Wigner 6j符号的结果，参见 Meserve 1973）。

**[待解决]** Berry相位和拓扑电荷需要从第一性原理重新推导。可能的路径：
1. 从15j符号的解析结构寻找非平凡相位
2. 从EPRL顶点振幅的复结构
3. 从历史路径积分的复耦合常数

**[重要警示]** 之前给出的$Q_{\text{topo}}/e = 0, 1/4, 3/4$缺乏严格推导基础，暂不采用。

### 4.2 $\alpha$的严格表达式与高精度验证

> **权威推导**：完整的推导链（含认识论标记、参考文献）参见 [../Foundations/附录A-α的严格推导.md](../Foundations/附录A-α的严格推导.md)。

#### 4.2.1 几何输入

EPRL相位来自正则4-单纯形的二面角。4-单纯形的二面角为：

$$\Theta = \arccos\left(\frac14\right)$$

**[定理4.3]** EPRL顶点振幅的Regge相为：

$$S_{\text{Regge}} = \sum_{f=1}^{10} j_f \Theta_f$$

对于j$_f$=1/2的各向同情况，总Regge相=10×$\Theta/2 = 5\Theta$(因为j$_f \cdot \Theta_f = (1/2)\Theta_f f是连续的二面角)。EPRL相位$\phi$是$S_{\text{Regge}} \bmod 2\pi$：

$$\phi = 5\arccos\left(\frac14\right) \bmod 2\pi$$

使用Chebyshev多项式$T_5(x) = 16x^5 - 20x^3 + 5x$：

$$\cos\phi = \cos(5\arccos(1/4)) = T_5(1/4) = \frac{61}{64}$$

这在域$\mathbb{Q}(\sqrt{15})$中是严格精确的代数数。

#### 4.2.2 精细结构常数的纯几何表达式

**[经验关系4.1]** 精细结构常数$\alpha$的纯几何表达式：

$$\boxed{\frac{1}{\alpha_0} = \frac{4\pi}{\sin^2\phi}}$$

代入$\sin^2\phi = 1 - \cos^2\phi = 1 - (61/64)^2 = 375/4096$：

$$\frac{1}{\alpha_0} = \frac{4\pi}{375/4096} = \frac{16384\pi}{375} \approx 137.258$$

**[数值事实4.1]** (使用mpmath进行500位精度计算)

| 量 | 值 |
|----|----|
| $\phi$ | 0.307404 rad |
| $\cos\phi$ | 61/64 = 0.953125 |
| $\sin^2\phi$ | 375/4096 ≈ 0.09155 |
| $1/\alpha_0$ | 137.258 |
| $1/\alpha_{\text{PDG}}$ | 137.035999084(21) |
| **相对偏差** | **0.16%** |

**[注]** 偏差0.16%需要解释。可能来源：
1. 历史路径积分的累积效应
2. 高阶几何修正
3. 其他未考虑的物理机制

**HPI修正与偏差分析**：参见 [../Level2/03-二级质变理论.md](../Level2/03-二级质变理论.md) §5（α的几何锁定与HPI修正）和 [../Foundations/附录A-α的严格推导.md](../Foundations/附录A-α的严格推导.md) §Step 6。

#### 4.2.3 偏差分析

**[数值事实4.2]** 0.16%偏差是真实的，来自历史路径积分的修正，需要从第一性原理推导。

**[开放问题4.1]** 0.16%偏差的来源：

1. **历史路径积分修正**: 边界标记涨落、拓扑缺陷、多路径干涉的综合作用

## 5 夸克作为闭合核的边与SU(3)颜色

### 5.1 本体论修正

**[核心修正]** 与标准夸克模型中将夸克视为"部分破缺态"不同，闭合核理论赋予夸克不同的本体论地位：

> **夸克不是再生产产物，而是闭合核(质子)内部intertwiner的边。**

这一修正的几何基础：

- 4价intertwiner有4条边，每条j=1/2
- 其中3条是"活跃"的——对应3个夸克(cf. 质子 = uud)
- 第4条是再生产连接边——当质子再生产时，此边打开以释放轻子对

**[推导5.1]** 禁闭的本体论解释：

1. 单条边j=1/2不是SU(2)规范不变量 → 不能作为物理态
2. 只有4条边的整体J=0态是物理的 → 质子
3. "拆开质子" = 打破边之间的SU(2)约束 = 移除整个intertwiner结构
4. 所需能量 = $\infty$ (因为需要破坏SU(2)规范不变性)

这与QCD禁闭的动力学解释一致，但更为本质：禁闭不是胶子动力学的结果，而是"夸克作为边"的数学定义的必然推论。

### 5.2 SU(3)颜色的几何起源

**[定理5.1]** 3条活跃边的置换群为S$_3$ (6阶)。S$_3$是SU(3)的Weyl群：

$$W(SU(3)) = N(T)/T = S_3$$

其中$T$是SU(3)的极大环面(对角矩阵)，$N(T)$是其在SU(3)中的正规化子。

**[推导5.2]** S$_3$的3维置换表示的不可约分解：

$$3 = 1 \oplus 2 \quad (\text{作为 S}_3 \text{表示})$$

- **1(平凡表示)**: 3条边的全对称组合——色单态(物理强子)
- **2(标准表示)**: 混合对称组合——色"双态"(但SU(3)将其提升为三维基本表示)
- 全反对称组合(对应S$_3$的符号表示)出现在color singlet中

色空间维度: $\mathbf{3}_{\text{color}}$来自3条活跃边的酉空间。

**[数值事实5.1]** S$_3$的生成元(置换矩阵)在SU(3)中的嵌入：

$$P_{12} = \begin{pmatrix}0&1&0\\1&0&0\\0&0&1\end{pmatrix},\quad
P_{123} = \begin{pmatrix}0&0&1\\1&0&0\\0&1&0\end{pmatrix}$$

验证: $P_{12}^2 = I$, $P_{123}^3 = I$, $P_{12}P_{123}P_{12} = P_{123}^2$。

S$_3$投影算符的秩:

$$\frac{1}{6}\sum_{g\in S_3} g = \text{色单态投影}, \quad \text{rank}=1$$

$$I - P_{\text{trivial}} = \text{色双态投影}, \quad \text{rank}=2$$

**[物理含义]** SU(3)颜色**不是**外加的对称性，而是3条活跃边的离散置换对称性(S$_3$)在量子涨落下的"软化"。边的量子化取向涨落将离散S$_3$提升为连续SU(3)。这正是规范理论中Weyl群提升为完整Lie群的典型过程。

### 5.3 夸克味

**[提案5.1]** 夸克味(flavor)对应边的不同"电荷状态"——即边的弱同位旋T$_3$分量：

- 上分量(T$_3=+1/2, Q=+2/3$): u-like夸克
- 下分量(T$_3=-1/2, Q=-1/3$): d-like夸克

这与轻子完全平行: $(\nu_e, e^-)$与$(u, d)$的电荷差来自超荷$Y$的不同。

**[开放问题5.1]** 三代夸克质量($u/d$, $c/s$, $t/b$)如何从边的激发谱涌现？目前缺乏定量机制。

**[开放问题5.2]** CKM混合矩阵——不同代夸克之间的弱耦合——如何从边的味振动中推导？已知CKM与PMNS有本质不同，暗示轻子(再生产产物)和夸克(边)的混合机制不同。

## 6 粒子本体论总表与关键预测

### 6.1 综合粒子分类

**[推导6.1]** 闭合核理论中所有粒子类型的本体论分类：

```
                   闭合核 = 质子 = 4价j=1/2 intertwiner
                   ╱       ╲
                  ╱         ╲
    3条活跃边 = 夸克     再生产连接边 → 再生产
    (永远禁闭)             ╲         ╱
                            ╲       ╱
                          3维配对态 → 轻子
                          (3代, 自由粒子)

   对称性的联络:
   ╱    ╲   ╲
  U(1)  SU(2) SU(3)
(Berry相) (破缺) (边置换)
   │      │      │
   γ_B   W±,Z    8g
```

| 粒子类型      | 本体论地位      | 物理表现              | 自由度              |
| --------- | ---------- | ----------------- | ---------------- |
| **夸克**    | 闭合核的边      | 永远禁闭，不自由          | 3色×2味×3代         |
| **轻子**    | 再生产产物      | 自由传播              | 2弱分量×3代          |
| **规范玻色子** | 对称性的联络     | 传递相互作用            | U(1)×SU(2)×SU(3) |
| **质子(p)** | **唯一的闭合核** | 绝对稳定(在CPT对称性下)    | —                |
| **中子(n)** | 近似闭合核      | 节拍不匹配，$\tau=880$s | —                |
| **其他强子**  | 闭合核的组合     | 强或弱衰变             | —                |
|           |            |                   |                  |

### 6.2 定理性预测

**[预测6.1]** 不存在第四代轻子。配对态空间维数=3是SU(2)⊗SU(2)分解的必然结果。实验支持：LEP精确测量$Z\to\nu\bar\nu$宽度给出$N_\nu = 2.984\pm0.008$。

**[预测6.2]** 不存在自由夸克。夸克是intertwiner的边，不具备SU(2)规范不变性。所有实验从未观察到孤立夸克。

**[预测6.3]** 不存在SU(3)$_c$以外的颜色群扩展到SU(N$>3$)。$\mathbf{3}$来自3条活跃边的空间，$\mathbf{8}$来自伴随表示(胶子)。无"第四色"。

**[预测6.4]** 质子绝对稳定。若闭合核满足四大判据，则操作闭合不能被破坏——不存在质子衰变通道。实验下限$\tau_p > 2.4\times10^{34}$ yr(C宫水)与预测一致。

**[预测6.5]** 中微子为Majorana粒子。配对态$|\psi_k\rangle$(self-conjugate under charge conjugation)来自CG系数的对称性。$0\nu\beta\beta$实验待验证。

**[预测6.6]** $\alpha$不是自由参数。表达式(经验关系4.1)给出纯几何值$1/\alpha_0=16384\pi/375\approx 137.258$，与实验值偏差0.16%。物理意义：$\alpha$由4-单纯形几何($\sin^2\phi$)纯几何决定，0.16%偏差来自历史路径积分修正。
