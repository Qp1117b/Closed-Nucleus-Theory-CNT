# 第一部分：基本概念与公理基础

## §1.1 孤立再生产的定义

**[定义]** 孤立再生产是闭合核在无网络连接时进行的再生产模式。

在范畴论层面（参见 [CategoryTheory.lean](file:///d:\WorkSpace\物理\闭合核理论\CNTFormal\CNTFormal\CategoryTheory.lean)）：

   - 每个闭合核存在至少一个幂等的再生产态射
   - 在形式不变阶段，再生产是稳定的

2. **公理1（五判据）**：包含条件债务态射 ε: S→S，¬IsIso ε

## §1.2 一级质变的定义

**[定义]** 一级质变是闭合核从孤立再生产到网络化再生产的结构性跃迁。

| 时期 | 再生产模式 | 系统状态 |
|------|-----------|----------|
| 质变前 | 孤立再生产 | 单一闭合核，自我维持 |

| 质变后 | 网络化再生产 | 多个闭合核通过穿刺连接 |

## §1.3 一级质变的触发机制

**[推导]** 一级质变的触发由历史面标记饱和驱动：

1. 每次再生产对历史面（对径面）产生标记
2. 标记累积达到阈值时触发质变
3. 阈值由4-单纯形的几何结构决定

# 第三部分：历史面标记饱和

## §3.1 标记增量 m_k

**[定义]** m_k 是第 k 次再生产对历史面（对径面）的标记量。候选形式：

| 选项 | 数学形式 | 语义 |
|------|----------|------|
| M1: 常数1 | m_k = 1 | 每次再生产标记一次 |
| M3: 线性k | m_k = k | 历史权重线性增长 |
| M6: Catalan(k) | m_k = C_k | 组合结构增长 |
| M7: 幂等迹 | m_k = Tr(μ_k) - Tr(μ_{k-1}) | 范畴论语义 |

**[推导] 第一优先选择**：M1（常数1）。
- 最小假设：每次再生产的历史贡献相同
- 最简单的数学形式
- 与公理4的幂等性一致（每次再生产结构相同）

## §3.2 阈值 Θ

**[定义]** Θ 是历史面标记饱和的临界值，即触发一级质变所需的最小标记总量。

候选形式：

| 选项 | 数值 | 几何来源 |
|------|------|----------|
| T1: 边数3 | Θ = 3 | 历史面（对径面）三角形边数 |
| T5: 三角形数3 | Θ = 3 = T₂ | 三角形数序列的第2项 |

**[推导] 第一优先选择**：Θ = 3。
- 历史面是三角形（3条边）
- 标记饱和意味着"每条边都被标记"
- 这是最简单的几何解释

## §3.3 标记累积方程

**[定义]** 累积标记量 M_k = Σ_{i=1}^k m_i（简单求和 C1）。

对于 M1（m_k = 1）和 Θ = 3：
```
M_k = k
M_{N_c} = N_c = Θ = 3
→ N_c = 3
```

**[定理3.1]** M1 + T1 给出 N_c = 3。

**[推导]** 验证：3次再生产后，历史面的3条边各被标记一次 → 饱和。

**[定义]** 第 k 步再生产的粒子数增量：

```
N_{k+1} = N_k + n_k
```

其中 n_k ≥ 0（量变累积假设 accumulation_nonnegative）。

**[推导]** 第一优先选择 n_k = 1（N1：常数1）：

```
N_{k+1} = N_k + 1
```

这是最简单的再生产微分方程，满足：
1. 量变累积假设（n_k ≥ 0）
2. 公理4（每次再生产相同）
3. 最小参数复杂度

**[定义]** 总能量子数 Nn = Σ_{k=1}^{N_c} n_k。

对于 n_k = 1 且 N_c = 3：

```
Nn = 1 + 1 + 1 = 3
```

**[定理4.1]** Nn = N_c = 3（当 n_k = 1 时）。

# 第五部分：序参量与临界条件

## §5.1 一级序参量 Φ

**[定义]** 一级序参量 Φ(N, f) 标记系统距离质变临界点的距离。

**[推导]** 候选形式（Φ2：比例形式）：

```
Φ(N, f) = N · f - α · Nn
```

其中 α 是耦合常数。

**[简化形式]** 取 α = 1：
```
Φ(N, f) = N · f - Nn
```

## §5.2 临界条件

**[定义]** 临界条件：Φ(N_c, f) = 0。

**[推导]** 代入 Φ 的定义：
```
N_c · f = Nn = 3
```

因为 N_c, f, Nn ∈ ℕ⁺（正整数约束），所以临界对 (N_c, f) 满足 N_c · f = 3。

**[定理5.1]** 可能的临界对：
```
(3, 1) 或 (1, 3)
```

取最小 N_c 解：N_c = 1, f = 3 或 N_c = 3, f = 1。

**[定理5.2]** 根据 N_c = 3（来自标记饱和）→ f = Nn / N_c = 3 / 3 = 1。

## §5.3 自旋量子化

**[定理5.3]** 临界条件 N_c · f = 奇数 ⇒ 最小网络自旋 j_min = 1/2。
完整网络自旋谱 j ∈ {0, 1/2, 1, 3/2, 2, ...}。
对于给定临界对 (N_c, f)，网络自旋值 j = (N_c·f)/2，给出半整数自旋谱。
整数自旋 j ∈ ℕ 在一级质变后的网络化再生产过程中涌现。

**[证明]**
```
N_c · f = 3 → 奇数
在 interwiner 空间中，奇数乘积对应 j 半整数
Catalan(2) = 2 → 分裂数为 2
j = 奇数/2 = 1/2
```

**[推论5.4]** 自旋量子化是临界条件的拓扑推论。

## §5.4 临界对的9重简并
