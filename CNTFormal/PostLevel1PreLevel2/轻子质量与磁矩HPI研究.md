### 三次质量方程

轻子的3个激发态共享同一个三次方程:

```
d m³ - b m² + m - a = 0

其中:
- a = m_e m_μ m_τ / (m_e m_μ + m_e m_τ + m_μ m_τ) ≈ m_e
- b = (m_e + m_μ + m_τ) / (m_e m_μ + m_e m_τ + m_μ m_τ) ≈ 1/m_μ
- d = 1 / (m_e m_μ + m_e m_τ + m_μ m_τ) ≈ 1/(m_μ m_τ)
```

因式分解:
```
(m - m_e)(m - m_μ)(m - m_τ) = 0
```

## 1. 摘要

本理论提出了一种结合圈量子引力 (LQG) 与弦论的新框架，用于严格推导轻子（τ, μ, e）的质量谱。理论基于**质子闭合核**（4-valent SU(2) Intertwiner）的几何结构，通过**能量生产弦**的动力学过程，解释了质量的起源。

核心突破在于：
1.  **质量即概率**：轻子质量正比于弦在固定生产周期内成功生长并闭合的概率。
2.  **单一总周期**：系统具有唯一的特征时间尺度，与粒子种类无关。
3.  **无拟合参数**：所有关键参数（g_s, C_μ, C_e）均由第一性原理（电磁耦合、弦统计、几何拓扑）严格导出。

---

## 2. 核心物理图像

### 2.1 质子闭合核结构
质子闭合核是一个由 3 条硬夸克边（自旋 j=1/2）组成的自旋网络三角形结构。该结构存在一个**拓扑缺陷**——缺失第 4 条边（轻子边）。

### 2.2 弦的"生长-闭合"机制
当外部能量注入闭合核时，触发以下过程：
1.  **生长**：能量在缺陷顶点处转化为一根开弦，开始向另一顶点延伸。
2.  **路径选择**：弦根据量子态选择生长路径。
    *   **τ 子**：最短路径（直接连接），生长时间极短。
    *   **μ 子**：中等路径（绕行），生长时间中等。
    *   **e 子（基态）**：最长路径（复杂拓扑/双圈缠绕），生长时间最长。
3.  **闭合**：弦端点相遇并融合，形成闭合弦（Closed String）。
4.  **脱离**：闭合弦不再受核束缚，作为自由轻子粒子脱离。

---

## 3. 动力学机制：单一总周期

### 3.1 总生产周期 (T_cycle)
系统存在一个固定的**平均生产周期**，这是质子闭合核的内在时钟，与最终产出的粒子种类无关。

*   **定义**：光穿过质子核特征尺度所需的时间。
*   **推导**：由最快闭合的 τ 子（基准态）决定。
    T_cycle = ħ/(m_τ c²)
*   **数值**：
    T_cycle ≈ 3.704 × 10⁻²⁵ s

### 3.2 质量生成原理
在固定周期 T_cycle 内，弦能否成功闭合取决于其路径的复杂性。质量被定义为**该周期内弦成功闭合的概率权重**与基准能量的乘积：

m_n = m_τ × P_n

*   P_n：第 n 种态在周期内成功闭合的概率。
*   **物理意义**：
    *   概率越高（τ），能量释放越集中 → **质量大**。
    *   概率越低（e），能量分布越稀疏 → **质量小**。

## 4. 参数推导（待重写）

### 4.1 弦耦合常数 (g_s)
g_s 是弦在生长过程中单次尝试成功的概率幅，由电磁耦合和弦统计熵决定。

*   **电磁耦合因子 (α)**：弦端点连接带电夸克，涉及电荷重分布，概率幅 ∝ √α。
*   **弦统计因子 (e)**：弦的振动模式贡献额外相空间体积（态密度增强），因子为 e。
*   **总概率**：
    g_s² = α × e  ⇒  g_s = √(αe)
*   **数值**：
    g_s = √(1/137.036 × 2.71828) ≈ 0.1408

### 4.2 几何修正因子 (C_n)

#### (1) μ 子修正因子 (C_μ)
*   **物理来源**：通道简并度。
*   **推导**：闭合核有 3 条夸克边，弦有 3 个可能的连接通道。
    C_μ = 3

#### (2) e 子修正因子 (C_e)
*   **物理来源**：构型比 × 几何截面压制。
*   **构型比 (15/16)**：4 条边共 2⁴=16 种自旋构型，排除 1 种全对称基态，剩余 15 种允许 e 态闭合。
*   **几何截面 (π/4)**：圆形弦截面与方形核背景的面积比。
*   **总因子**：
    C_e = 15/16 × π/4 = 15π/64 ≈ 0.7363
