### 三次质量方程

轻子的3个激发态共享同一个三次方程:

```
d m³ - b m² + m - a = 0

其中:
- a = m_e m_μ m_τ / (m_e m_μ + m_e m_τ + m_μ m_τ) ≈ m_e
- b = (m_e + m_μ + m_τ) / (m_e m_μ + m_e m_τ + m_μ m_τ) ≈ 1/m_μ
- d = 1 / (m_e m_μ + m_e m_τ + m_μ m_τ) ≈ 1/(m_μ m_τ)
```

因式分解:
```
(m - m_e)(m - m_μ)(m - m_τ) = 0
```

### 为什么是一边再生产？

| 模型 | 组合数 | 匹配3代？ | 物理合理性 |
|------|--------|----------|-----------|
| 1条边激发 | C(4,1)=4 | ❌ | 低 |
| **2条边配对** | **C(4,2)/2=3** | **✓** | **高** |
| 3条边激发 | C(4,3)=4 | ❌ | 中 |
| 4条边激发 | C(4,4)=1 | ❌ | 低 |

**注意**: 虽然组合分析显示两边配对有3种方式，但物理图像上，轻子作为单边的3个激发态更自然：
- 夸克占据3条边 (3色)
- 轻子占据剩下1条边
- 轻子边与3条夸克边耦合，产生3个激发态
