## 2. 系数C的严格分解结构

### 2.1 主分解

卡西米尔力的标准形式为：

$$\frac{F}{A} = C \cdot \frac{\hbar c}{a^4}$$

在CNT中，系数C的结构为：

$$C = G_{\text{geom}} \times g_{cm} \times N_{\text{mode}}$$

其中：
- **G_geom**：纯几何因子（由4-单纯形结构决定，无动力学）
- **g_cm**：引力耦合常数（曲率到质量的映射强度）
- **N_mode**：模式计数因子（由边界条件和网络动力学决定）

### 2.2 素因数分解

240的素因数分解是唯一的：240 = 2⁴ × 3 × 5。

这给出两种天然分组：

**分组A（cot²θ分解）**：
$$C_0 = \frac{\pi^2}{240} = \frac{\pi^2}{16 \times 15} = \frac{\pi^2}{16} \cdot \frac{1}{15} = \frac{\pi^2}{16} \cdot \cot^2\theta$$

物理意义：
- π²/16：连续极限下的模式积分因子
- cot²θ = 1/15：4-单纯形的纯几何耦合因子

**分组B（S₅分解）**：
$$C_0 = \frac{\pi^2}{240} = \frac{\pi^2}{2 \times 120} = \frac{\pi^2}{2 \cdot 5!} = \frac{\pi^2}{2 \cdot |S_5|}$$

物理意义：
- 2：偏振自由度
- 120 = 5!：正4-单纯形对称群S₅的阶数

**两种分组通过数字8关联**：16 = 2×8, 120 = 15×8。8 = 2³ 恰好是3个可见面法向量方向的组合数（每个面法向量可取±方向）。

### 2.3 分组A的优越性

分组A（cot²θ分解）在物理上更优，原因：
1. cot²θ是直接可计算的几何量，不需额外假设
2. 240 = 2⁴×(3×5) 是素因数分解，16和15是2⁴和3×5的自然分组
3. 分组B中240 = 2×120不是素因数分解，120 = 2³×3×5 仍包含混合因子

**本文后续推导以分组A为主方案。**

## 3. 几何因子cot²θ=1/15的三种推导路径

### 3.1 路径A：曲率-质量投影效率（最物理）

**核心思想**：引力是曲率对质量的直接吸引。虚核携带曲率（铰链面上的ε₀），板中闭合核具有质量（可见面上的信息压缩刚性）。曲率从铰链面传播到可见面的效率取决于两个面的相对取向。

**几何事实**：在4-单纯形中，任意两个三角形面之间的二面角等于 arccos(1/4) 或其补角。二面角θ决定了相邻四面体之间的"开口"大小。

**投影论证**（尝试性）：
- 当铰链面与可见面平行时 → 曲率传递最大（效率=1）
- 当垂直时 → 曲率传递为零（效率=0）
- 平均传递效率 = ⟨cos²(相对角)⟩

**猜测**：在所有可能的相对取向下，平均传递效率恰好为 cot²θ = 1/15。这意味着约6.67%的虚核曲率能有效转化为对板中质量的引力。

**严格性等级**：⚠️ 猜测。需要证明：
1. 相对角的分布函数
2. 平均值恰好是cot²θ
3. 归一化常数恰好是1

### 3.2 路径B：虚核涌现的各向异性权重

**核心思想**：虚核不是各向同性涌现的。边界条件（导体板的存在）打破了空间的各向同性，只有某些朝向的虚核对两板之间的引力有贡献。

**猜测**：虚核的共享四面体τ必须与板面满足特定几何关系才能产生有效引力。这个选择规则的几何权重 ∝ cot²θ。

**严格性等级**：⚠️ 猜测。需要HPI方程给出的朝向分布函数。

### 3.3 路径C：信息压缩投影因子

**核心思想**：引力本质上是信息表面压缩。cot²θ 是信息从虚核铰链面投影到板闭合核可见面的效率因子。

$$投影效率 = \frac{\text{有效投影面积}}{\text{源面积}}$$

这与CNT的核心洞察——"质量是信息表面压缩刚性的量度"——最为自洽。

**严格性等级**：⚠️ 猜测。但此路径与CNT本体论框架最一致。

### 3.4 Cot²θ与α₀的关系

注意到一个重要事实：cot²θ = 1/15 和 1/α₀ = 16384π/375 都来自4-单纯形的三角函数有理值：
