## 一、量纲错误（DIMENSIONAL ERRORS）

### 1.1 【严重】§3.1 物理量维度表错误

**位置**：§3.1 的物理量定义表格

**错误描述**：
- 表格中 Φ(r) 标注为量纲 [1]（无量纲），但根据 E7 的定义 Φ(r) = -Φ_M/(4πr)，其中 Φ_M = γ₀ N_M = [T⁻¹]，因此 Φ(r) = [L⁻¹ T⁻¹]，不是无量纲
- 表格中 G_M(r) 标注为 [L⁻¹ T⁻¹]，但根据 E7 的定义 G_M = Φ_M/(4πr²) = [L⁻² T⁻¹]

**修复**：修正表格中 Φ(r) 和 G_M(r) 的量纲标注

---

### 1.2 【致命】E8 量纲不自洽——δ 不是无量纲量

**位置**：§3.2 方程 E8

**错误描述**：
E8 定义 δ = μ_eff G_M R_m / (k_B T_int)，其中：
- [μ_eff] = [M]
- [G_M] = [L⁻² T⁻¹]（由 E7）
- [R_m] = [L]
- [k_B] = [M L² T⁻² K⁻¹]
- [T_int] = [K]

[δ] = [M][L⁻² T⁻¹][L] / ([M L² T⁻² K⁻¹][K]) = [L⁻³ T]

**δ 不是无量纲量！** 但 E8 中 δ 出现在指数函数 e^(δ cos θ) 中，指数必须是纯数。这是一个根本性的量纲错误。

数值验证：代入 CNT 典型参数，δ ≈ 10⁴⁰（而非论文声称的 10⁻⁵³），完全违反弱场条件 δ ≪ 1。

**物理根源**：G_M 被定义为信息通量面密度 [L⁻² T⁻¹]，但 E8 的玻尔兹曼因子需要的是能量，而非通量。G_M 应该定义为加速度 [L T⁻²] 才能与 μ_eff 结合产生能量。

**修复方案**：
在熵力范式中，将信息场重新定义为加速度场。引入 CNT 基本长度 ℓ₀ 和频率 ν 作为转换因子：

$$\boxed{\mathcal{G}_M(r) = \frac{c^2 \ell_0}{\nu} \cdot \frac{\gamma_0 M}{4\pi \mu_0 r^2}}$$

当 γ₀ = ν 时简化为：

$$\boxed{\mathcal{G}_M(r) = \frac{c^2 \ell_0 M}{4\pi \mu_0 r^2}}$$

[G_M] = [L² T⁻²][L]/([M][L²]) × [M] = [L T⁻²] ✓（加速度）

此时 δ = μ_eff G_M R_m/(k_B T_int) = [M][L T⁻²][L]/([M L² T⁻²]) = [1] ✓

---

### 1.3 【致命】熵力范式 G_eff 量纲错误

**位置**：§4.5 的 G_eff 表达式

**错误描述**：
论文给出的 G_eff^(ent) = T_scr ς β γ₀ / (16π T_int μ₀)

量纲分析：[T_scr ς β γ₀ / (T_int μ₀)] = [1][1][1][T⁻¹]/([1][M]) = [M⁻¹ T⁻¹]

但牛顿引力常数 G 的量纲为 [L³ M⁻¹ T⁻²]。

**熵力范式的 G_eff 量纲与牛顿 G 不一致！** 缺少 [L³ T⁻¹]。

**对比**：发射范式 G_eff^(emi) = κ_geo γ₀ c/(12π μ₀ ρ_max)，其量纲为 [T⁻¹][L T⁻¹]/([M][L⁻²]) = [L³ M⁻¹ T⁻²] ✓，量纲正确。

**修复**：若采用 1.2 节的修复方案，G_eff^(ent) 变为：
$$\boxed{G_{\text{eff}}^{\text{(ent)}} = \frac{T_{\text{scr}} \varsigma \beta c^2 \ell_0}{16\pi T_{\text{int}} \mu_0}}$$

量纲：[1][1][1][L² T⁻²][L]/([1][M]) = [L³ M⁻¹ T⁻²] ✓

---

### 1.4 【致命】方程 (7.1) 量纲错误

**位置**：§7.2 的 κ_geo 与熵力范式参数的映射关系

**错误描述**：
κ_geo = 3 T_scr ς β ρ_max / (4 c T_int)

量纲分析：[T_scr ς β ρ_max / (c T_int)] = [1][1][1][L⁻²]/([L T⁻¹][1]) = [L⁻³ T]

但 κ_geo 是无量纲量。**此方程量纲错误。**

**修复**：采用 1.2 节的修复后，新映射关系为：
$$\boxed{\kappa_{\text{geo}} = \frac{3 T_{\text{scr}} \varsigma \beta c \ell_0 \rho_{\max}}{4 T_{\text{int}} \gamma_0}}$$

量纲：[1][1][1][L T⁻¹][L][L⁻²]/([1][T⁻¹]) = [1][L² T⁻¹][L⁻²]/([T⁻¹]) = [1] ✓

## 二、数值估计错误（NUMERICAL ERRORS）

### 2.1 【严重】§3.2 G_M 数值估计严重错误

**位置**：§3.2 第2段 "作为数值量级，对典型星系参数..."

**错误描述**：论文声称 G_M ~ 10⁻²⁰（无量纲，按原表格）或 ~ 10⁻²⁰ m⁻² s⁻¹（按公式实际量纲）

实际计算（取 M_gal = 10¹¹ M⊙，r = 10 kpc）：
G_M = γ₀ M/(4π μ₀ r²) = 6.77×10³⁸ × 1.99×10⁴¹ / (4π × 4.99×10⁻¹² × (3.086×10²⁰)²) ≈ 2.3×10⁴⁹ m⁻² s⁻¹

**论文的估计偏差达 69 个数量级！**

**修复**：删除此数值估计，或用正确的数值替换。注意：在采用 1.2 节的修复后，G_M 的量纲和数值都会改变。

---

### 2.2 【严重】§3.2 δ 数值估计严重错误

**位置**：§3.2 第2段 "由此 δ ~ 10⁻⁵³..."

**错误描述**：给定 G_M 的正确值 2.3×10⁴⁹，δ ≈ 8.1×10⁴⁰（远大于 1），而非论文声称的 10⁻⁵³。

弱场近似 δ ≪ 1 完全不成立。论文的数值估计偏差达 93 个数量级。

**修复**：在采用 1.2 节的修复后重新计算。若 G_M 重新定义为加速度场，δ 的量级需要重新评估。

## 三、概念矛盾与逻辑错误

### 3.1 【严重】§4.1 量纲矛盾陈述

**位置**：§4.1 开头 "G_M(r) 具有量纲 [L⁻² T⁻¹]（等价地理解为 [L⁻¹ T⁻¹]）"

**错误描述**：[L⁻² T⁻¹] 和 [L⁻¹ T⁻¹] 是不同量纲，不可能"等价"。这是明显的量纲矛盾。

**修复**：删除此句或修正为正确的量纲说明。

---

### 3.2 【严重】§4.5 β 量纲矛盾

**位置**：§4.5 的推导中

**错误描述**：β = μ_eff/μ₀ 被定义为无量纲耦合比（β ~ O(1)），但后文又暗示 β 具有量纲。这是前后矛盾。

**修复**：明确 β 是无量纲量，β = μ_eff/μ₀ ∈ [0,1]。

---

### 3.3 【中等】§9.4 自我否定

**位置**：§9.4 最后一段

**错误描述**：论文承认简单幂律 Θ_sat = η^n 无法产生平坦旋转曲线（需要 n = -1/2，但 n 必须为正）。这实际上否定了 §9.2-9.3 的 MOND 解释框架。

**修复**：明确标注此问题，指出需要更复杂的 Θ_sat(η) 函数形式或不同的 η(r) 径向轮廓。

---

### 3.4 【中等】§10.6 循环论证

**位置**：§10.6 独立量纲常数表

**错误描述**：论文承认 ρ_max 由 ℓ_P 决定，ℓ_P 需要 G_exp 确定，而 G_eff 是推导目标。这构成了循环论证。

**修复**：论文已如实标注，但应更明确地说明这是当前理论的局限性，而非可接受的推导路径。

## 四、公式引用错误

### 4.1 【中等】§10.3 公式引用错误

**位置**：§10.3 开头 "由(4.11)"

**错误描述**：公式 (4.11) 是发射范式的 G_eff 公式，但 §10.3 的上下文是熵力范式的 G_eff 数值验证。编号混淆。

**修复**：将 (4.11) 改为正确的引用，或在 §10.3 中说明使用的是发射范式的公式。

## 五、关联论文（信息密度涌现...）问题

### 5.1 【中等】ν 选择与 Gibbons-Hawking 温度不一致

**位置**：关联论文 §5.3-5.4

**错误描述**：Gibbons-Hawking 温度 T_GH = ħ H₀/(2π k_B)，对应的特征角频率为 ω = H₀/(2π)。标准频率 ν = ω/(2π) = H₀/(4π²)。但论文取 ν = H₀/(2π)（即取角频率而非频率），且论文自身已标注此问题。

**修复**：论文已明确标注此假设，但建议在摘要和结论中同样提及此不确定性。

---

### 5.2 【中等】BTFR 预言偏差 53%

**位置**：关联论文 §8.4.4

**错误描述**：CNT 预言 BTFR = 72.0 M⊙/(km/s)⁴，观测值为 47 ± 6，偏差 53%。论文将此归因于模型修正，但未提供修正方案。

**修复**：标注为需要进一步研究的开放问题。
