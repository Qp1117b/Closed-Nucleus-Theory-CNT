# 第一部分：几何基础——4-单纯形的严格构造

## §1.1 顶点坐标

**[推导]** 正则4-单纯形 $\Delta_4$ 嵌入 $\mathbb{R}^5$ 的超平面 $\sum x_i = 0$。5个顶点：

$$\mathbf{v}_i^{(5)} = \mathbf{e}_i \in \mathbb{R}^5, \quad i = 0,1,2,3,4$$

法向量 $\mathbf{n} = \frac{1}{\sqrt{5}}(1,1,1,1,1)^T$，投影 $P = I_5 - \mathbf{n}\mathbf{n}^T$。

**[推导]** $P$ 的秩为4，取其非零特征值对应的特征向量构成 $\mathbb{R}^4$ 基底 $B \in \mathbb{R}^{5\times 4}$（$B^T B = I_4$）：

$$\mathbf{v}_i^{(4)} = B^T \mathbf{e}_i \in \mathbb{R}^4$$

**[推导] 边长**：$|\mathbf{v}_i - \mathbf{v}_j|^2 = 2$（$i \neq j$），故 $\ell = \sqrt{2}$。

## §1.2 组合结构

**[推导]** 由 $\binom{5}{k+1}$：

| $k$-面 | 数量 | 记号 |
|--------|------|------|
| 顶点 | $v = 5$ | $V$ |
| 边 | $e = 10$ | $E$ |
| 三角形面 | $f = 10$ | $F$ |
| 四面体 | $t = 5$ | $T$ |

**[推导] 面对偶于边**：面 $(ijk)$ 对偶于边 $(lm)$ 当 $\{i,j,k\} \cap \{l,m\} = \emptyset$。

## §1.3 二面角与体积

**[定理1.1]** 正则 $n$-单纯形中，两个 $(n-1)$-面沿共享 $(n-2)$-面的二面角 $\vartheta_n$ 满足 $\cos\vartheta_n = 1/n$。

**[推导]** 对于 $n=4$：
$$\vartheta_4 = \arccos\left(\frac{1}{4}\right) \approx 75.5225^\circ \tag{1.1}$$

**[推导] 体积**（Sommerville 1958）：
$$V_4 = \frac{\sqrt{5}}{96}\ell^4, \quad V_3 = \frac{\sqrt{2}}{12}\ell^3, \quad A_2 = \frac{\sqrt{3}}{4}\ell^2 \tag{1.2}$$

# 第二部分：面bivector与Gram矩阵

## §2.1 面bivector的显式构造

**[定义]** 面 $F = (ijk)$ 的bivector：
$$\mathbf{B}_{ijk} = \mathbf{e}_{ij} \wedge \mathbf{e}_{ik}$$

其中 $\mathbf{e}_{ij} = \mathbf{v}_j^{(4)} - \mathbf{v}_i^{(4)}$。坐标表示：$B_{ijk}^{\mu\nu} = \frac{1}{2}(e_{ij}^\mu e_{ik}^\nu - e_{ij}^\nu e_{ik}^\mu)$。

**[定义] bivector内积**：
$$\langle B_1, B_2 \rangle = \frac{1}{2} B_1^{\mu\nu} B_{2,\mu\nu} = \frac{1}{2}\operatorname{Tr}(B_1^T B_2) \tag{2.1}$$

**[推导] 面bivector的范数**：
$$\|B_{ijk}\|^2 = \frac{1}{4}(|u|^2|v|^2 - (u\cdot v)^2) = \frac{1}{4}(4 - 1) = \frac{3}{4}$$

其中 $u = \mathbf{e}_{ij}$，$v = \mathbf{e}_{ik}$，$|u|^2 = |v|^2 = 2$，$u\cdot v = 1$（正三角形内角60°）。

## §2.2 Gram矩阵的谱

**[定理2.1]** 10个面bivector的Gram矩阵 $G_{ab} = \langle B_a, B_b \rangle$ 的秩为6。

**[数值事实]** 直接数值计算：
$$\lambda(G) = \left\{\frac{5}{4}, \frac{5}{4}, \frac{5}{4}, \frac{5}{4}, \frac{5}{4}, \frac{5}{4}, 0, 0, 0, 0\right\} \tag{2.2}$$

**[推导] 特征多项式**：
$$\det(\lambda I_{10} - G) = \lambda^4\left(\lambda - \frac{5}{4}\right)^6 \tag{2.3}$$

**[推导] 迹验证**：$\operatorname{Tr}(G) = 10 \times \frac{3}{4} = \frac{30}{4} = 7.5 = 6 \times \frac{5}{4}$ ✓

# 第三部分：Hodge分解——6 = 3 + 3

## §3.1 Hodge星算子

**[定义]** 在 $\mathbb{R}^4$（欧几里得符号）上：
$$(\star B)_{\mu\nu} = \frac{1}{2}\varepsilon_{\mu\nu}^{\quad\rho\sigma} B_{\rho\sigma} \tag{3.1}$$

**[定理3.1]** $\star^2 = \mathrm{id}_{\wedge^2}$（欧几里得符号）。

**[推导]** 自对偶（SD）和反自对偶（ASD）投影：
$$B^\pm = \frac{1}{2}(B \pm \star B), \quad \star B^\pm = \pm B^\pm \tag{3.2}$$

**[定理3.2]** $\wedge^2\mathbb{R}^4 = \wedge^2_+\mathbb{R}^4 \oplus \wedge^2_-\mathbb{R}^4$，$\dim\wedge^2_\pm = 3$。

## §3.2 SD bivector的3维坐标

**[推导]** 将10个SD bivector $B_a^+$ 映射到 $\mathbb{R}^6$（标准基 $\{B^{\mu\nu}: \mu<\nu\}$），然后通过SVD投影到3维SD子空间。得到10个3维向量 $\mathbf{v}_a \in \mathbb{R}^3$。

**[数值事实]** 每个SD bivector的范数：
$$\|\mathbf{v}_a\|^2 = \|B_a^+\|^2 = \frac{1}{2}\|B_a\|^2 = \frac{3}{8} \tag{3.3}$$

## §3.3 SD子空间中的角度结构

**[数值事实]** 10个向量 $\mathbf{v}_a$ 的两两夹角仅取四个值：

$$\cos\theta \in \left\{\pm\frac{1}{3}, \pm\frac{\sqrt{5}}{3}\right\} \tag{3.4}$$

对应角度：
$$\theta \approx \{70.53^\circ, 109.47^\circ, 41.81^\circ, 138.19^\circ\} \tag{3.5}$$

**[推导] 关键恒等式**：
$$\left(\frac{1}{3}\right)^2 + \left(\frac{\sqrt{5}}{3}\right)^2 = \frac{1}{9} + \frac{5}{9} = \frac{6}{9} = \frac{2}{3} \tag{3.6}$$

# 第四部分：紧框架定理——核心几何事实

## §4.1 自然算子 $M$

**[定义]** 在3维SD子空间上构造算子：
$$M = \sum_{a=1}^{10} |\mathbf{v}_a\rangle\langle\mathbf{v}_a| \tag{4.1}$$

即 $M_{\mu\nu} = \sum_a (\mathbf{v}_a)_\mu (\mathbf{v}_a)_\nu$。

**[定理4.1]（紧框架定理）**：
$$\boxed{M = \frac{5}{4} I_3} \tag{4.2}$$

**[数值事实]** 直接计算验证：$M$ 的三个特征值均为 $5/4 = 1.25$。

**[推导] 迹验证**：
$$\operatorname{Tr}(M) = \sum_a \|\mathbf{v}_a\|^2 = 10 \times \frac{3}{8} = \frac{15}{4} = 3 \times \frac{5}{4} \quad \checkmark \tag{4.3}$$

**[推导] 等价表述**：对任意 $\mathbf{x} \in \mathbb{R}^3$，
$$\sum_{a=1}^{10} \langle\mathbf{v}_a, \mathbf{x}\rangle^2 = \frac{5}{4}\|\mathbf{x}\|^2 \tag{4.4}$$

这是**紧框架（tight frame）**的定义性质，框架常数为 $5/4$。

## §4.2 归一化向量与球面2-设计

**[定义]** 归一化向量 $\hat{\mathbf{v}}_a = \mathbf{v}_a / \|\mathbf{v}_a\| = \mathbf{v}_a / \sqrt{3/8}$。

**[推导]** 归一化算子：
$$M_{\text{norm}} = \sum_{a=1}^{10} |\hat{\mathbf{v}}_a\rangle\langle\hat{\mathbf{v}}_a| = \frac{8}{3} M = \frac{10}{3} I_3 \tag{4.5}$$

**[定理4.2]（球面2-设计定理）**：10个归一化向量 $\{\hat{\mathbf{v}}_a\}$ 构成 $S^2$ 上的球面2-设计。

**[推导]** 球面 $t$-设计的定义：对任意次数 $\leq t$ 的多项式 $f$，
$$\frac{1}{N}\sum_{a=1}^N f(\hat{\mathbf{v}}_a) = \frac{1}{4\pi}\int_{S^2} f(\mathbf{u}) d\Omega \tag{4.6}$$

对于 $t=2$，只需验证一阶和二阶矩。一阶矩：$\sum_a \hat{\mathbf{v}}_a = 0$（由对称性）。二阶矩：由(4.5)，$\sum_a (\hat{\mathbf{v}}_a)_\mu (\hat{\mathbf{v}}_a)_\nu = \frac{10}{3}\delta_{\mu\nu}$，与球面积分 $\frac{1}{4\pi}\int u_\mu u_\nu d\Omega = \frac{1}{3}\delta_{\mu\nu}$ 一致（乘以 $N=10$ 得 $10/3$）。

**[数值事实] 四阶矩验证**：对任意单位向量 $\mathbf{x}$，
$$\sum_{a=1}^{10} \langle\hat{\mathbf{v}}_a, \mathbf{x}\rangle^4 = 2 = \frac{10}{5} \tag{4.7}$$

球面积分：$\frac{1}{4\pi}\int_{S^2} (\mathbf{u}\cdot\mathbf{x})^4 d\Omega = \frac{1}{5}$，乘以 $N=10$ 得 $2$。✓

## §4.3 紧框架的物理意义

**[推导]** $M \propto I$ 意味着：在**经典水平**上，3维SD子空间是**完全各向同性的**。没有任何方向是特殊的。

# 第五部分：三次型——偏离各向同性的结构

## §5.1 三次型 $C$ 的定义

**[定义]** 自然的三次型（cubic form）：
$$C_{\mu\nu\rho} = \sum_{a=1}^{10} (\mathbf{v}_a)_\mu (\mathbf{v}_a)_\nu (\mathbf{v}_a)_\rho \tag{5.1}$$

对应的齐次三次多项式：
$$C(\mathbf{x}) = C(\mathbf{x},\mathbf{x},\mathbf{x}) = \sum_{a=1}^{10} \langle\mathbf{v}_a, \mathbf{x}\rangle^3 \tag{5.2}$$

**[定理5.1]** $C \neq 0$。三次型非零。

**[数值事实]** $C$ 的Frobenius范数 $\|C\| = \sqrt{\sum_{\mu,\nu,\rho} C_{\mu\nu\rho}^2} \approx 0.8859$。

**[推导]** 为什么 $C \neq 0$？因为10个归一化向量构成球面2-设计但**不是**球面3-设计。球面3-设计要求三阶矩为零，但这里三阶矩非零。

## §5.1a 民主基底中的精确结构

**[推导]** 收缩张量 $T_{\mu\nu} = \sum_{\rho,\sigma} C_{\mu\rho\sigma} C_{\nu\rho\sigma}$ 的特征向量定义了自然的"民主基底"：

$$\lambda(T) = \{\lambda_0, \lambda_1, \lambda_1\}, \quad \lambda_0 \neq \lambda_1$$

非简并特征向量 $\mathbf{v}_{\text{dem}}$（民主方向）是 $S_5$ 不变的。两个简并特征向量张成2维子空间。

**[定理5.1a]（民主基底中的三次型结构定理）**：在民主基底 $\{\mathbf{v}_{\text{dem}}, \mathbf{v}_{2a}, \mathbf{v}_{2b}\}$ 中，三次型 $C$ 具有极其简单的结构：

$$\boxed{C(x_0, x_1, x_2) = a \cdot x_0^3 + b \cdot x_0 \cdot (x_1^2 + x_2^2)} \tag{5.2a}$$

其中 $x_0 = \langle\mathbf{v}_{\text{dem}}, \mathbf{x}\rangle$，$(x_1, x_2)$ 是2维子空间坐标。

**[数值事实]** 2维子空间内的纯三次分量（$C_{111}, C_{112}, C_{122}, C_{222}$）**精确为零**（在 $10^{-16}$ 精度内）。

**[推导]** 这意味着 $C$ 在2维子空间内具有 **SO(2) 对称性**：$C$ 只依赖于 $x_0$ 和 $r^2 = x_1^2 + x_2^2$，不依赖于 $(x_1, x_2)$ 的角度。

**[推导] 系数 $a$ 和 $b$ 的精确表达式**：

10个SD bivector沿民主方向的投影仅取两个值：
$$c_1 = -\sqrt{\frac{5+2\sqrt{5}}{40}}, \quad c_2 = -\sqrt{\frac{5-2\sqrt{5}}{40}} \tag{5.2b}$$

各有5个面取 $c_1$，5个面取 $c_2$。

验证：$5(c_1^2 + c_2^2) = 5 \cdot \frac{1}{4} = \frac{5}{4} = \operatorname{Tr}(M)/3$ ✓

**[定理5.1b]（系数精确表达式）**：
$$\boxed{a = C_{000} = 5(c_1^3 + c_2^3)} \tag{5.2c}$$

$$\boxed{b = 3C_{011} = \frac{15}{2}\left[c_1\left(\frac{3}{8} - c_1^2\right) + c_2\left(\frac{3}{8} - c_2^2\right)\right]} \tag{5.2d}$$

**[数值事实]**：
$$a = -0.583751565768443, \quad b = -0.816094087306058 \tag{5.2e}$$

$$\frac{b}{a} = 1.398016099934161 \tag{5.2f}$$

**[推导]** 这些系数**完全由4-单纯形几何决定**，不含任何自由参数。$\sqrt{5}$ 的出现源于4-单纯形体积 $V_4 = \sqrt{5}/96 \cdot \ell^4$。

## §5.2 三次型的收缩

**[定义]** 收缩张量：
$$T_{\mu\nu} = \sum_{\rho,\sigma} C_{\mu\rho\sigma} C_{\nu\rho\sigma} = (C \cdot C)_{\mu\nu} \tag{5.3}$$

**[数值事实]** $T$ 的特征值：
$$\lambda(T) = \{0.48877, 0.14800, 0.14800\} \tag{5.4}$$

两个简并，一个独立。这是 $S_5$ 对称性在3维表示上的自然结果。

## §5.3 三次型的Waring分解

**[定义]** 三次型 $C$ 的Waring分解：
$$C(\mathbf{x}) = \sum_{i=1}^{r} \lambda_i \langle\mathbf{w}_i, \mathbf{x}\rangle^3 \tag{5.6}$$

其中 $r$ 是Waring秩，$\mathbf{w}_i$ 是单位向量，$\lambda_i$ 是权重。

**[数值事实]（Waring秩）**：$C$ 的Waring秩 $r = 4$（非3）。

**[推导]** Waring秩为4（非3）意味着：三次型 $C$ 不能直接分解为三个"味方向"。

## §5.4 SO(2)对称性的物理意义

**[推导]** 三次型 $C$ 在2维子空间内的SO(2)对称性意味着：

> **在经典（$S_5$对称）水平上，三次型不区分2维子空间内的两个方向。**

**[推导]** 对称性破缺的完整图像：

2. **三次型 $C$ 的SO(2)结构**：破缺到民主方向 vs 2维子空间
3. **量子修正**：进一步破缺SO(2)

# 第六部分：4-单纯形中5个四面体的几何约束

## §6.1 组合结构

**[数值事实]** 4-单纯形$\Delta_4$的组合结构：
- 顶点数：5
- 边数：10
- 三角形面数：10
- 四面体数：5

**[推导]** 每个四面体有4个三角形面，每个三角形面被2个四面体共享：
$$5 \times 4 = 10 \times 2 = 20 \tag{6.1}$$

## §6.2 四面体的闭合约束

**[定义]** 对于每个四面体$t$，其4个面的面积矢量必须满足闭合约束：
$$\sum_{f \subset t} A_f \mathbf{n}_f = 0 \tag{6.2}$$

其中$A_f$是面$f$的面积，$\mathbf{n}_f$是单位法向量。

**[推导]** 每个四面体的自由度计数：
- 4个面，每个面有2个角度自由度（球面坐标）
- 闭合约束提供3个方程
- 每个四面体的剩余自由度：$4 \times 2 - 3 = 5$

## §6.3 形状匹配约束

**[定义]** 4-单纯形中，每个三角形面被2个四面体共享。这两个四面体在该面上的几何必须匹配：
$$A_f^{(t_1)} = A_f^{(t_2)}, \quad \mathbf{n}_f^{(t_1)} = -\mathbf{n}_f^{(t_2)} \tag{6.3}$$

**[推导]** 形状匹配约束计数：
- 10个共享面
- 每个面施加2个角度匹配约束（法向量方向匹配）
- 总约束数：$10 \times 2 = 20$

## §6.4 整体自由度计数

**[推导]** 4-单纯形的整体自由度：
- 5个四面体的总自由度：$5 \times 5 = 25$
- 形状匹配约束：$-20$
- 整体旋转自由度（SO(3)）：$-3$
- **预期剩余自由度**：$25 - 20 - 3 = 2$

**[推导]** 这2个自由度对应于4-单纯形的"形状模空间"。在Regge calculus中，4-单纯形由10个边长确定，但整体旋转和平移不变性减少了6个自由度，剩余$10 - 6 = 4$个自由度。然而，当面积固定后，剩余自由度进一步减少。

## §6.5 与Intertwiner空间的联系

**[推导]** 在量子几何中：
- 每个四面体对应一个SU(2) intertwiner
- 对于j=1的4价intertwiner，每个四面体的intertwiner空间维数为3
- 5个四面体的张量积空间维数：$3^5 = 243$

# 第七部分：组合恒等式

## §7.1 基本组合恒等式

**[推导]** 由4-单纯形的组合数据：
$$\boxed{\frac{f - k}{k} = \frac{10 - 6}{6} = \frac{4}{6} = \frac{2}{3}} \tag{6.1}$$

其中 $f=10$（面数），$k=6$（bivector空间维数）。

**[推导]** 等价形式：
$$\frac{f}{k} = \frac{5}{3}, \quad 3f = 5k = 30 \tag{6.2}$$

## §7.2 9/8与16/27的代数对偶结构

**[推导]** 两个关键数值因子的代数结构：

$$\frac{9}{8} = \frac{3^2}{2^3}, \quad \frac{16}{27} = \frac{2^4}{3^3} \tag{6.7}$$

**[定理6.1]（对偶乘积定理）**：
$$\boxed{\frac{9}{8} \times \frac{16}{27} = \frac{2}{3} = \frac{f-k}{k}} \tag{6.8}$$

**[推导] 用 $f=10$, $k=6$ 表示**：
$$\frac{9}{8} = \frac{f-1}{f-2} = \frac{9}{8} \tag{6.9a}$$

$$\frac{16}{27} = \frac{(f-2)^2}{4(k/2)^3} = \frac{64}{4 \times 27} = \frac{16}{27} \tag{6.9b}$$
