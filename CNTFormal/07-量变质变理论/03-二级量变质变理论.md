# 第一部分：量变质变等级体系

## §1.1 量变质变等级的定义

**[定义]** 在CNT框架中，"量变质变"不是连续的过程，而是结构性的跃迁。根据公理5（量变质变形式新立），每一次量变质变产生新的结构形式S_new ≠ S_old。

**[定义] 量变质变等级体系**：

| 等级 | 名称 | 触发条件 | 涌现结构 | 对应物理 |
|------|------|----------|----------|----------|
| L0 | 基础量变质变 | 物质自我差异化 → 操作闭合形成 | 闭合核本身 | 核子存在 |
| L1 | 一级量变质变 | 历史面标记饱和 Φ(N_c, f_c)=0 | c, l_min, j=1/2 | 网络化（辐射速度、网络自旋） |
| L2 | 二级量变质变 | 网络闭合后 Logistic 饱和 | α, e, U(1) gauge | 电磁涌现 |
| L3 | 三级量变质变 | （待研究） | （待研究） | 弱力涌现？ |

## §1.2 一级量变质变的回顾

**[推导]** 一级量变质变（见 Level1Transition.lean）的核心结果：

1. **临界条件**：Φ(N_c, f_c) = 0，等价于 N_c · f_c 为奇数
2. **涌现量**：
   - 自旋：j = 1/2
   - 光速：c = √2 · ℓ₀ · f_c（量纲 [L·T⁻¹]）
   - 最小距离：l_min = √2 · ℓ₀（量纲 [L]）

3. **物理解释**：孤立再生产 → 网络化再生产。闭合核之间通过穿刺（puncture）建立网络连接。

## §1.3 二级量变质变的定义

**[定义]** 二级量变质变是一级量变质变后，网络化再生产达到 Logistic 饱和时，体系经历的第二次结构性跃迁。其核心特征是：

1. ℤ₂ 交换对称性（经典核标记交换）量子化提升为 U(1) 规范对称性
2. 精细结构常数 α 被几何锁定
3. 电荷量子化作为 U(1) 拓扑分类涌现
4. SO(3) 空间对称性破缺为 SO(2)（U(1)）

# 第二部分：一级量变质变后的状态

## §2.1 网络化再生产

**[推导]** 一级量变质变后，闭合核通过穿刺构成网络。每个闭合核的3个可见面（见 KernelPerspective.lean T33）各承载一个穿刺。

**[定义] 穿刺（Puncture）**：闭合核之间通过历史面（对径面）的对偶边建立的量子连接。每个穿刺携带自旋 j=1/2。

## §2.2 交换对称性 ℤ₂

**[推导]** 一级量变质变后的网络再生产方程：

```
N_A(k+1) = (1 + J₀) · N_A(k) + J₀ · N_B(k) + f
N_B(k+1) = (1 + J₀) · N_B(k) + J₀ · N_A(k) + f
```

其中 J₀ 是交换耦合，N_A, N_B 是两个耦合的闭合核系统的穿刺计数。

**[定理2.1]** 此方程在 A ↔ B 交换下不变。

**[证明]** 交换 A ↔ B 后，方程变为：
```
N_B(k+1) = (1 + J₀) · N_B(k) + J₀ · N_A(k) + f
N_A(k+1) = (1 + J₀) · N_A(k) + J₀ · N_B(k) + f
```
与原方程组相同。□

**[推论2.2]** 系统具有 ℤ₂ = {+1, -1} 经典离散对称性。ℤ₂ 的作用是交换两个核的身份标记。

## §2.3 总穿刺数的 Logistic 动力学

**[推导]** 总穿刺数 N_tot = N_A + N_B 的演化方程：
```
N_tot(k+1) = N_A(k+1) + N_B(k+1)
            = (1+J₀)N_A + J₀N_B + f + (1+J₀)N_B + J₀N_A + f
            = (1+2J₀)(N_A+N_B) + 2f
            = (1+J) · N_tot(k) + 2f
```

其中 J = 2J₀ = γ·Γ（总耦合常数）。

**[推导] Logistic 饱和**：当穿刺网络达到一定密度后，进一步穿刺受限于面容量。用 Logistic 方程描述：

```
N_{k+1} = N_k + J · N_k · (1 - N_k/N_max) + 2f
```

其中 N_max 是最大穿刺数，由网络拓扑决定。

## §2.4 交换耦合 J₀ 的几何起源

**[推导]** 交换耦合 J₀ 来自4-单纯形几何与热力学自洽性：

```
γ = ln2/(π√3) ≈ 0.127（Immirzi参数，由准黑洞热力学自洽性 S_LQG = S_BH 确定）
Γ = 3/5（几何因子 = 可见面数/总顶点数）
J = γ · Γ = 3ln2/(5π√3)
J₀ = J/2 = 3ln2/(10π√3)
```

**[数值事实]**：
```
γ = ln2/(π√3) ≈ 0.127384
Γ = 3/5 = 0.6
J = γ·Γ ≈ 0.076430
J₀ = J/2 ≈ 0.038215
√(3/5)/10 ≈ 0.077460
```

**[推导] 验证**：J₀ = 3ln2/(10π√3) ≈ 3×0.6931/(10×3.1416×1.7321) ≈ 0.038215 ✓

# 第三部分：Logistic 饱和与临界穿刺数 N_c

## §3.1 Logistic 饱和拐点

**[定理3.1]** Logistic 方程 N_{k+1} = N_k + J·N_k·(1 - N_k/N_max) 的拐点位于 N = N_max/2。

**[证明]** Logistic 映射的增长率 dN/dk 在 N = N_max/2 处取最大值：
```
f(N) = N + J·N·(1 - N/N_max)
df/dN = 1 + J·(1 - 2N/N_max)
d²f/dN² = -2J/N_max < 0
令 df/dN = 0 → 1 + J·(1 - 2N/N_max) = 0
但在地图形式下，拐点直接由二次项对称性给出：N = N_max/2。
```

在连续极限下，Logistic 微分方程 dN/dt = r·N·(1 - N/K) 的拐点在 N = K/2。□

**[定义]** 临界穿刺数 N_c = N_max/2。

## §3.2 N_c 的确定

**[推导]** 从物理要求出发：N_c 必须由几何纯数决定。

> **【循环论证声明】** 公式 $\alpha=\|C\|^2/(\pi\cdot N_c)$ 与 $N_c=\|C\|^2/(\pi\cdot\alpha_{\text{exp}})=34.23$ 构成循环论证：用实验 $\alpha$ 确定 $N_c=34$，再声称 $N_c$ "锁定"了 $\alpha$。当前 $N_c=34$ 并非从网络拓扑独立推导，而是从实验 $\alpha$ 反推。在 $N_c$ 被独立推导之前，本理论的"无自由参数预测 $\alpha$"声明不成立。此为开放问题（见 §9.1）。

N_c 通过精细结构常数锁定：
```
α = ||C||² / (π · N_c)
```

其中 ||C|| ≈ 0.8859 是三次型 C（见 01号文档 §5.1）的 Frobenius 范数。

使用实验值 α_exp = 1/137.035999084：
```
N_c = ||C||² / (π · α_exp) = 0.8859² / (π / 137.036) ≈ 34.23
```

**[定理3.2]** N_c 必须是整数（穿刺数只能取整数）。最接近的整数是 N_c = 34。

**[数值事实]**：
```
||C||² = 0.8859² ≈ 0.784819
π · α_exp = π / 137.036 ≈ 0.022925
N_c(精确) = 0.784819 / 0.022925 ≈ 34.23
N_c(取整) = 34
偏差 = (34 - 34.23)/34.23 ≈ -0.67%
```

## §3.3 N_max 的拓扑约束

**[推导]** N_max = 2·N_c = 68。

网络拓扑解释：
- 每个闭合核有3个可见面 → 最多3个穿刺
- 每个穿刺连接2个核的面 → 总穿刺数上限受限于面的总数
- 需要更多研究来确定68的精确拓扑来源

**[开放问题]** N_max = 68 的确切网络拓扑推导。

# 第四部分：ℤ₂ → U(1) 的提升

## §4.1 经典 vs 量子对称性

**[定理4.1]** 经典 ℤ₂ 交换对称性在量子化后提升为 U(1) 连续对称性。

**[推导]** 论证步骤：

1. **经典层面**：再生产方程在 A↔B 交换下不变 → ℤ₂ 对称性
2. **量子化**：每个穿刺携带自旋 j=1/2 → 态空间是 SU(2) 表示
3. **穿刺网络形成后**：大量 j=1/2 穿刺耦合 → 集体激发
4. **Goldstone 模式**：连续对称性破缺 SO(3)→SO(2) 产生无质量模式
5. **提升机制**：ℤ₂ 作为 U(1) 的离散子群，在红外极限下涌现连续的 U(1)

**[定理4.2]** U(1) 涌现的拓扑分类：
```
π₁(U(1)) = π₁(S¹) = ℤ
```
→ U(1) 的不可约表示由整数 n ∈ ℤ 标记：ρ_n(θ) = e^{inθ}
→ 电荷量子数 n = ±1（电子/正电子），n = ±1/3, ±2/3（夸克，待进一步研究）

## §4.2 U(1) 涌现的时空结构

**[推导]** U(1) 规范场 A_μ 从穿刺网络的集体激发涌现：
- 时空是连续的背景（见最终本体论），一级量变质变后闭合核网络嵌入其中
- U(1) 联络 A_μ 描述穿刺相位的空间变化
- F_μν = ∂_μ A_ν - ∂_ν A_μ 是电磁场张量

## §4.3 Nambu-Goldstone 模式

**[推导]** SO(3) → SO(2) ≅ U(1) 的对称性破缺产生：
```
dim SO(3) - dim SO(2) = 3 - 1 = 2
```
2个 Nambu-Goldstone 玻色子。但这与物理上只有一个光子（2个偏振态）一致：
- 2个NG玻色子 = 光子的2个横向偏振态（在Lorentz规范下）

# 第五部分：精细结构常数的几何锁定

> **互补视角**：本节从二级量变质变（U(1)涌现、临界穿刺数N_c）的视角锁定α。裸几何推导（4-单纯形→EPRL相位→1/α₀ = 16384π/375）参见 [../01-公理体系/04-附录A-α的严格推导.md](../01-公理体系/04-附录A-α的严格推导.md)。两者互补但不重复。

## §5.1 锁定公式的推导

> **【循环论证声明】** 公式 $\alpha=\|C\|^2/(\pi\cdot N_c)$ 与 $N_c=\|C\|^2/(\pi\cdot\alpha_{\text{exp}})=34.23$ 构成循环论证：用实验 $\alpha$ 确定 $N_c=34$，再声称 $N_c$ "锁定"了 $\alpha$。当前 $N_c=34$ 并非从网络拓扑独立推导，而是从实验 $\alpha$ 反推。在 $N_c$ 被独立推导之前，本理论的"无自由参数预测 $\alpha$"声明不成立。此为开放问题（见 §9.1）。

**[定理5.1]**（精细结构常数的几何锁定公式）：
```
α = ||C||² / (π · N_c)
```

其中 ||C|| 是三次型 C 的 Frobenius 范数，N_c = 34 是临界穿刺数。

**[推导]** 从序参量出发：

1. 三次型 C 描述了 SD 子空间中4-单纯形面的非各向同性分布（见01号文档 §5.1）
2. C 的 Waring 秩 = 4 = 3(空间) + 1(电荷)
3. 在二级量变质变时，SO(3)→SO(2)破缺后，C 投影到 U(1) 方向
4. 耦合常数的强度由 ||C||² 测度
5. π 因子来自球面立体角归一化（与 4π 因子同源）
6. 分母 N_c 来自网络穿刺数的量子统计平均

**[数值事实]**：
```
α(几何) = 0.8859² / (π · 34) = 0.784819 / 106.814 = 0.00734752
1/α(几何) = 136.10
1/α(实验) = 137.036
偏差 = (136.10/137.036 - 1) × 100% = -0.68%
```

## §5.2 裸α与实验α的关系

**[推导]** EPRL 相位给出的裸 α：
```
α⁻¹_裸 = 4π/sin²(φ) = 16384π/375 ≈ 137.258277
```

## §5.3 与EPRL相位的自洽性

**[定理5.2]** 两个独立的 α 推导——EPRL 相位和二级量变质变锁定——在实验精度内自洽。

**[推导]**：

路径1（EPRL 相位）：
```
α⁻¹(EPRL) = 4π/sin²(φ) = 16384π/375 = 137.258277
```

# 第六部分：电荷e的涌现

## §6.1 U(1)表示的拓扑分类

**[定理6.1]** 电荷量子化的拓扑起源：U(1) 的不可约表示由 winding 数分类。

**[推导]**：

2. 基本群 π₁(S¹) = ℤ
3. 不可约表示 ρ_n: U(1) → U(1)，ρ_n(e^{iθ}) = e^{inθ}，n ∈ ℤ
4. 量子数 n 对应电荷（以电子电荷为基本单位）
5. 电子：|n| = 1
6. 核穿刺具有分数电荷（n = ±1/3, ±2/3），来自 SU(3)_c 色禁闭的残留效应

## §6.2 电荷的有量纲表达式

**[推导]** CNT 框架下，α 是无量纲的基本物理常数。电荷 e 不是纯数，需要通过 α、ħ、c、ε₀ 来确定其有量纲的值。

惯例（在各单位制中）：
```
α = e²/(4π ε₀ ħ c)    （SI 单位）
```

→ e² = 4π ε₀ ħ c · α

α 由几何锁定公式确定后，e 的量纲由 ħ、c、ε₀ 的组合给出。

量纲验证：
```
[e²] = [ε₀] · [ħ] · [c]
     = (C²/N·m²) · (J·s) · (m/s)
     = C²/(kg·m/s²·m²) · (kg·m²/s) · (m/s)
     = C² ✓
[e] = C ✓
```

**[数值事实]**：
```
ħ = 1.054571817 × 10⁻³⁴ J·s
c = 2.99792458 × 10⁸ m·s⁻¹
α_exp = 1/137.035999084
ε₀ = 8.8541878188 × 10⁻¹² F/m (C²/N·m²)

e² = 4π × 8.854 × 10⁻¹² × 1.055 × 10⁻³⁴ × 2.998 × 10⁸ × (1/137.036)
   ≈ 2.566 × 10⁻³⁸ C²
e  ≈ 1.6022 × 10⁻¹⁹ C ✓（与 CODATA 一致）
```

## §6.3 CNT框架下α与e的关系

**[推导]** CNT 框架下的观点：
- α 是由几何锁定的唯一基本无量纲物理常数
- α 定义的是电磁相互作用的**强度**
- e 的 SI 值由 α、ħ、c、ε₀ 的组合决定，是派生量
- ħ 和 c 作为基本物理常数出现在所有电动力学公式中
- ε₀ 和 μ₀ 是 SI 单位的约定，c² = 1/(ε₀ μ₀)

# 第七部分：与朗道二级相变理论的对应

## §7.1 朗道理论的基本框架

**[回顾]** 朗道二级相变理论（Landau, 1937）：

1. 引入序参量 φ：在对称相中 φ = 0，在破缺相中 φ ≠ 0
2. 自由能展开：F = F₀ + a(T-T_c)φ² + bφ⁴ + ...
3. 临界行为：φ ∝ (T_c - T)^{1/2}（平均场指数 β = 1/2）

## §7.2 CNT二级量变质变的朗道对应

**[定理7.1]** CNT二级量变质变可以映射到朗道范式：

| 朗道理论 | CNT 二级量变质变 |
|----------|-------------|
| 温度 T | 穿刺数 N（控制参数） |
| 临界温度 T_c | 临界穿刺数 N_c = 34 |
| 序参量 φ | U(1) 相位序参量（穿刺网络的集体相位） |
| 对称相 φ=0 | ℤ₂ 对称相（穿刺网络未形成连续对称性） |
| 破缺相 φ≠0 | U(1) 对称相（SO(3)→SO(2)破缺） |
| 自由能 F | 有效作用量 S_eff（HPI 路径积分） |

**[推导]** CNT 的有效自由能（HPI 框架）：

```
S_eff(N, θ) = ∫ d⁴x [a(N-N_c) |∂θ|² + b|∂θ|⁴ + |F_μν|²]
```

其中 θ 是 U(1) 相位场，F_μν = ∂_μA_ν - ∂_νA_μ，A_μ = ∂_μθ。

## §7.3 临界指数

**[猜想]** CNT 二级量变质变的临界指数：

1. 序参量指数 β：穿刺网络密度 ρ ∝ |N - N_c|^β
   - 平均场预测：β = 1/2
   - 这个指数需要从 Logistic 方程严格推导

2. 关联长度指数 ν：穿刺网络关联长度 ξ ∝ |N - N_c|^{-ν}
   - 需要确定 U(1) 理论的上临界维度

3. 比热指数 α_s：有效作用量的二阶导数

**[开放问题]** 从 CNT 第一性原理推导临界指数。
