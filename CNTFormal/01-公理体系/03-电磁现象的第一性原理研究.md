$$\text{4-单纯形几何} \to \text{内禀挠率} \to \text{S}_5\text{分解}: \mathbf{10}=\mathbf{1}\oplus\mathbf{4}\oplus\mathbf{5} \to \begin{cases} \text{均匀挠率 }V_1 \to \text{电荷} \to \alpha \\ \text{中性挠率 }V_{4\oplus5} \to \text{中微子} \end{cases}$$

# 第一部分：4-单纯形几何——电磁现象的发源地

## §1.1 二面角 $\Theta = \arccos(1/4)$

**[定理]** 正则4-单纯形的二面角为 $\Theta = \arccos(1/4) \approx 75.5225^\circ$。这是纯几何事实，由4-单纯形的 $S_5$ 对称性唯一确定。

有两条等价路径到达这一结论：

- **路径A（纯几何）**：通过顶点坐标法或Gram矩阵法直接计算两个四面体胞腔沿共享三角形面的夹角。参见 [01-4-单纯形几何理论.md](01-4-单纯形几何理论.md) §1.3。
- **路径B（Bianchi一致性检验）**：假设每条边承载内禀挠率，离散Bianchi恒等式要求面上的曲率holonomy为π-旋转，解出 $\Theta = \arccos(1/4)$。参见 [03-精细结构常数的第一性原理推导.md](03-精细结构常数的第一性原理推导.md) §0。

路径B不是 $\Theta$ 的独立推导（因为其几何不变量 $D_s=1, \Delta=2\sqrt{5}/(3\sqrt{3})$ 来自4-单纯形几何），而是**自洽性检验**，确认了内禀挠率假设与4-单纯形几何的一致性。

## §1.2 内禀挠率——第零公理

**[假设]** 4-单纯形的每条边 $e$ 携带内禀挠率：
$$\mathbf{T}_e = \frac{\Delta\theta_e}{\ell_e} \cdot \hat{\mathbf{n}}_e, \qquad \Delta\theta_e = \Theta = \arccos(1/4)$$

其中 $\ell_e = \sqrt{2}\,\ell_0$ 是边长，$\hat{\mathbf{n}}_e$ 是边在自对偶(SD) bivector子空间中的投影轴。

**[推导]** 内禀挠率不是由物质引入的——它是时空基本单元的固有属性。每条边可形象地理解为微型"螺丝"，10条边的螺丝共同构成4-单纯形的内禀扭曲场。参见 [00-内禀挠率_第零公理.md](00-内禀挠率_第零公理.md)。

**公理Z0（内禀挠率）**：正4-单纯形的每条边携带非零内禀挠率 $\mathbf{T}_e$。相邻面的标架差由挠率holonomy决定：
$$\mathbf{n}_2 = \exp(\mathbf{T}_e\ell_e) \cdot \mathbf{n}_1$$

## §1.3 S₅对称性与边空间的不可约分解

**[定理]** 10维边空间（每条边上的挠率分量）在 $S_5$ 作用下分解为不可约表示：
$$\mathbf{10} = \mathbf{1} \oplus \mathbf{4} \oplus \mathbf{5}$$

- **$\mathbf{1}$（均匀挠率）**：所有10条边上挠率相等，$S_5$ 作用下不变
- **$\mathbf{4}$**：标准表示，对应Young图 $[4,1]$
- **$\mathbf{5}$**：对应Young图 $[3,2]$

**[推导]** 此分解通过构造 $S_5$ 的正交投影算子 $P_1, P_4, P_5$ 严格验证。验证标准：$P_i P_j = \delta_{ij} P_i$, $P_1+P_4+P_5 = I_{10}$, $\operatorname{rank}(P_1)=1$, $\operatorname{rank}(P_4)=4$, $\operatorname{rank}(P_5)=5$。

这一分解是电磁现象与中性现象分化的**群论基础**：均匀挠率 $V_1$ 对应电荷模式，$V_{4\oplus 5}$ 对应中微子模式。

## §1.4 Cartan曲率算子与挠率→曲率的效率度量

**[定理1.1]** 边-面关联矩阵 $E$（$E_{fe}=1$ 若边 $e \subset$ 面 $f$）定义的Cartan曲率算子 $M = E^T E$ 的本征值为：
$$\boxed{\lambda(M) = \{9, 4, 4, 4, 4, 1, 1, 1, 1, 1\}}$$

**[推导]** 物理意义：给定边挠率 $T_e$，面曲率 $R_f = \sum_e E_{fe} T_e$，总曲率平方 $S = \sum_f R_f^2 = \sum_{e,e'} T_e M_{ee'} T_{e'}$。本征值直接度量了不同挠率模式产生曲率的效率。

**[推导] 解析证明**：$M$ 与 $S_5$ 对易，由Schur引理在每个不可约子空间上正比于恒等算子。均匀向量 $\mathbf{u}=(1,\ldots,1)$ 给出 $\lambda_1=9$（每条边在3个面中 + 与6条相邻边共享面 = 9）。利用 $\operatorname{Tr}(M)=30$, $\operatorname{Tr}(M^2)=150$ 解出 $\lambda_4=4$, $\lambda_5=1$。

**[定理1.2]** 曲率效率比：
$$\frac{S_{\text{uniform}}}{S_{V_5}} = \frac{9}{1} = 9, \qquad \frac{S_{\text{uniform}}}{S_{V_4}} = \frac{9}{4} = 2.25$$

**物理意义**：均匀挠率（电荷模式）产生曲率的效率是 $V_5$ 中性模式的**9倍**！这是电磁相互作用比弱相互作用强得多的**几何根源**。

# 第三部分：电荷的几何起源——均匀挠率

## §3.1 电荷 = 均匀挠率分量

**[推导]** $S_5$ 分解 $\mathbf{10} = \mathbf{1} \oplus \mathbf{4} \oplus \mathbf{5}$ 给出了电荷与中性的群论区分：

- **电荷** $\longleftrightarrow$ 均匀挠率 $V_1$（$\mathbf{1}$ 维平凡表示）：所有10条边挠率相等 → 总挠率 $\neq 0$
- **中性** $\longleftrightarrow$ $V_{4\oplus5}$ 挠率（$\mathbf{4}\oplus\mathbf{5}$ 维）：总挠率 $= 0$（无电荷）

**[推导]** 均匀挠率的Cartan本征值为 $\lambda_1=9$，中性模式 $\lambda_5=1$。曲率效率比 $9:1$ 直接解释了为什么电磁相互作用远强于弱相互作用。

## §3.2 裸电荷

**[推导]** 从EPRL相位直接导出：
$$e_0 = \sin(5\Theta) = \frac{5\sqrt{15}}{64} \approx 0.302577$$

在SI单位制：
$$e_{0,\text{SI}} = \sqrt{4\pi\varepsilon_0\hbar c \cdot \alpha_0} = \sqrt{\frac{375}{4096} \cdot 4\pi\varepsilon_0\hbar c} \approx 1.6009 \times 10^{-19}\,\text{C}$$

**[开放问题]** $e_0 = \sin(5\Theta)$ 这一物理对应关系（电荷-挠率相位）能否从第一原理严格证明？目前它是通过 $\alpha = e^2/(4\pi)$ 逆推的对应关系。

## §3.3 电荷手性

**[推导]** 内禀挠率的旋向决定电荷符号：每条边上的挠率 $\mathbf{T}_e$ 有旋向（右旋/左旋）。10条边的总旋向决定了电荷的符号。在4-单纯形中，面的遍历方向定义了挠率holonomy的符号，进而区分正/负电荷。

## §3.6 中微子的整体无挠率机制——V₄⊕V₅定理（v2.0 核心成果）

- $\sum T_e < 10^{-14}$（所有试验）
- $\|T\| \sim 2.4-2.9$

**[推论3.1]** 中微子对应 $V_4 \oplus V_5$ 挠率构型：
- 10条边的挠率不全相等（非均匀分布）

- Cartan本征值 $\lambda_{4,5} = 4, 1$（远小于电磁通道的 $\lambda_1 = 9$）

**[推论3.2] 电磁-中性分化的群论根源**：
$$\text{电荷: } V_1 \leftrightarrow \sum T_e \neq 0 \leftrightarrow \text{最大曲率} \leftrightarrow \text{电磁相互作用}$$
$$\text{中性: } V_4 \oplus V_5 \leftrightarrow \sum T_e = 0 \leftrightarrow \text{弱曲率} \leftrightarrow \text{弱相互作用}$$

在v8f开发过程中，发现此前所有PMNS脚本（v5-v8e）中的T张量计算存在**索引顺序错误**：

错误代码：
```python
C_murho = sum(face[:, mu] * face[:, nu] * face[:, rho])  # C_{μ,ν,ρ}
C_nurho = sum(face[:, nu] * face[:, mu] * face[:, rho])  # C_{ν,μ,ρ}
T[mu, nu] += C_murho * C_nurho  # 仅对ρ求和，缺少σ指标
```

正确代码：
```python
C[mu, nu, rho] = sum(face[:, mu] * face[:, nu] * face[:, rho])  # C_{μνρ}
T[mu, nu] = sum_{ρ,σ} C[mu, ρ, σ] * C[nu, ρ, σ]  # 对ρ,σ双指标求和
```

修正后，9:3:1模式恢复：$\kappa_0:\kappa_1:\kappa_2 = 2{,}035{,}383 : 685{,}411 : 229{,}002 \approx 8.89:2.99:1$，与calc_931_grouptheory_v1.py完全一致。
